# LobeVidol 用户指南

## 文档贡献方式

1. 克隆仓库

```bash
git clone git@github.com:v-idol/docs.vidol.chat.git
```

2. 安装依赖

```bash
pnpm install
pnpm i -g mintlify
```

3. 启动文档

```bash
mintlify dev
```

4. 提交 PR
