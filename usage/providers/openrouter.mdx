## 概述

OpenRouter 是一个强大的 AI 模型聚合平台，支持包括 GPT-4、Claude 2、LLaMA 2 等多种顶级模型。通过本指南，你可以轻松地在 LobeVidol 中接入这些模型。OpenRouter 作为一个模型聚合服务平台，为用户提供了以下优势：

  - 支持多种顶级 AI 模型，包括:
    - OpenAI 系列 (GPT-3.5/4)
    - Anthropic 系列 (Claude2、Claude Instant)
    - Meta 的 LLaMA 2
    - Google 的 PaLM 系列
  - 统一的 API 接口
  - 灵活的计费方式
  - 部分模型提供免费额度


## 配置步骤

### 1. 注册 OpenRouter 账号并登录

- 访问 [OpenRouter.ai](https://openrouter.ai/) 并创建一个账号
- 支持两种登录方式：
  - Google 账号登录
  - MetaMask 钱包登录

<Frame>
  <img src="https://oss.vidol.chat/assets/4dc5f7b6823b2836c4e69dd692889a9d.webp" alt="注册 OpenRouter" />
</Frame>

### 2. 创建 API 密钥

1. 进入 `Keys` 菜单或直接访问 [OpenRouter Keys](https://openrouter.ai/keys)
2. 点击 `Create Key` 按钮
3. 在弹出框中设置：
   - 为 API 密钥命名（如 "LobeVidol Key"）
   - `Credit limit` 可选填写（留空表示不限制）

<Note type="warning">
  创建成功后请立即复制并安全保存 API 密钥，因为它只会显示一次。如果遗失需要重新创建新的密钥。
</Note>

<Frame>
  <img src="https://oss.vidol.chat/assets/914d26be0d8dd7e2bce85ecad8192a53.webp" alt="创建 OpenRouter Key" />
</Frame>

### 3. 充值信用额度

1. 进入 `Credit` 菜单或访问 [OpenRouter Credit](https://openrouter.ai/credits)
2. 点击 `Manage Credits` 进行充值
3. 在 [模型价格页面](https://openrouter.ai/models) 查看各个模型的具体费率

<Note>
  OpenRouter 提供部分免费模型，无需充值即可使用。建议先试用免费模型，熟悉后再考虑付费模型。
</Note>

<Frame>
  <img src="https://oss.vidol.chat/assets/307d22deca37827f552eebe22bccb19b.webp" alt="充值 OpenRouter 信用额度" />
</Frame>

### 4. 在 LobeVidol 中配置 OpenRouter

1. 打开 LobeVidol 的 `设置` 界面
2. 找到 `语言模型` 部分的 `OpenRouter` 设置项
3. 启用 OpenRouter 并填入 API 密钥
4. 在助手设置中选择需要使用的 OpenRouter 模型

## 使用建议

### 模型选择
- 建议从免费模型开始尝试
- 根据具体需求选择合适的模型
- 注意不同模型的计费标准

### 费用控制
- 设置合理的 Credit limit
- 定期检查使用额度
- 关注账单变化

### 性能优化
- 选择延迟较低的模型
- 适当调整上下文长度
- 优化提示词以提高效率
