# Introduction à MikuMikuDance (MMD)

MikuMikuDance (abrégé MMD) est un logiciel gratuit de création d'animations 3D développé par Yu Higuchi. Il permet aux utilisateurs d'importer des modèles créés avec d'autres logiciels de modélisation 3D, tels que les chanteurs virtuels VOCALOID comme Hatsune Miku, pour réaliser des animations. Le terme "MMD" est également couramment utilisé pour désigner les œuvres animées produites avec ce logiciel.

## Instructions d'utilisation

- Les fichiers de danse MMD sont généralement soigneusement réalisés par des passionnés ou générés à l'aide d'équipements de capture de mouvement.
- Ces fichiers sont souvent soumis à des règles d'utilisation strictes, telles que l'interdiction d'utilisation commerciale ou de redistribution.
- Bilibili dispose d'une chaîne MMD dédiée : [Chaîne MMD - Bilibili](https://www.bilibili.com/v/channel/1126/?tab=featured)
- La communauté MMD offre une riche ressource de fichiers de danse, disponibles sur Bilibili ou sur des sites spécialisés en modèles.
