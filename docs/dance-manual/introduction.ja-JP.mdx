# MikuMikuDance (MMD) の紹介

MikuMikuDance（略称MMD）は、樋口優によって開発された無料の3Dアニメーション制作ソフトウェアです。このソフトウェアを使用すると、初音ミクなどのVOCALOIDバーチャルシンガーが作成したモデルを他の3Dモデリングソフトウェアからインポートしてアニメーションを制作できます。「MMD」は、このソフトウェアを使用して制作されたアニメーション作品を指すこともあります。

## 使用方法

- MMDのダンスファイルは、通常、愛好者によって丁寧に制作されるか、モーションキャプチャデバイスを使用して生成されます。
- これらのファイルには、商業利用や二次配布を禁止するなど、厳格な使用規則があります。
- Bilibiliには専用のMMDチャンネルがあります: [MMD-ビリビリチャンネル](https://www.bilibili.com/v/channel/1126/?tab=featured)
- MMDコミュニティには、Bilibiliや専門のモデルサイトで入手できる豊富なダンスファイルリソースがあります。
