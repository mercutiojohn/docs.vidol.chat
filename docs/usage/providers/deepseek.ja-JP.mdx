# LobeVidolでのDeepSeekの使用

<img
src="https://oss.vidol.chat/docs/2024/12/58a05d76bb89eeb59e238998b69c7059.png"
alt="LobeVidolでのDeepSeekの使用"
/>

[DeepSeek](https://www.deepseek.com/)は、先進的なオープンソースの大規模言語モデル（LLM）です。最新バージョンのDeepSeek-V2は、アーキテクチャとパフォーマンスが大幅に最適化されており、トレーニングコストは42.5%削減され、推論コストは93.3%削減されています。

このドキュメントでは、LobeVidolでDeepSeekを使用する方法を説明します。

## ステップ1：DeepSeek APIキーの取得

1. まず、[DeepSeek](https://platform.deepseek.com/)のオープンプラットフォームに登録してログインする必要があります。

<Info>現在、新規ユーザーには500Mトークンの無料クレジットが提供されます。</Info>

2. `API keys`メニューに移動し、`API Keyの作成`をクリックします。

<img
src="https://oss.vidol.chat/assets/498856598fdba60af6593408142db837.webp"
alt="DeepSeek APIキーの作成"
/>

3. ポップアップダイアログにAPIキーの名前を入力します。

<img
src="https://oss.vidol.chat/assets/89820732dcb092627c07d24143a37f60.webp"
alt="DeepSeek API名の入力"
/>

4. 取得したAPIキーをコピーして安全に保存します。

<img
src="https://oss.vidol.chat/assets/9f3d6287b883576a6b9cf67e28a6f43e.webp"
alt="DeepSeek APIキーの保存"
/>

<Warning>
  キーは一度しか表示されないため、安全に保管してください。誤って失った場合は、新しいキーを作成する必要があります。
</Warning>

## ステップ2：LobeVidolでDeepSeekを設定

1. LobeVidolの`アプリ設定`画面にアクセスします。
2. `言語モデル`の下にある`DeepSeek`の設定項目を見つけます。

<img
src="https://oss.vidol.chat/assets/ee9802f91be4d6f1b686825b2433d75d.webp"
alt="DeepSeek APIキーの入力"
/>

3. DeepSeekを開き、取得したAPIキーを入力します。
4. アシスタントのためにDeepSeekモデルを選択し、対話を開始します。

<img
src="https://oss.vidol.chat/assets/e84d84f68b6d6a8b3262dbcef2eded28.webp"
alt="DeepSeekモデルの選択"
/>

<Warning>
  使用中にAPIサービスプロバイダーに料金を支払う必要がある場合がありますので、DeepSeekの関連料金ポリシーを参照してください。
</Warning>

これで、LobeVidolでDeepSeekが提供するモデルを使用して対話を行うことができるようになりました。
