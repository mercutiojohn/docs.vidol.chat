# Utiliser Tongyi Qianwen dans LobeVidol

<img
src="https://oss.vidol.chat/docs/2024/12/83156a545234ce9264a97bb2991cfbbb.png"
alt="Utiliser Tongyi Qianwen dans LobeVidol"
/>

[Tongyi Qianwen](https://tongyi.aliyun.com/) est un modèle de langage à grande échelle développé par Alibaba Cloud, doté de puissantes capacités de compréhension et de génération du langage naturel. Il peut répondre à diverses questions, créer du contenu écrit, exprimer des opinions, rédiger du code, etc., et joue un rôle dans plusieurs domaines.

Ce document vous guidera sur la façon d'utiliser Tongyi Qianwen dans LobeVidol :

<Steps>
  <Step title="Activer le service de modèle DashScope">
    - Accédez et connectez-vous à la plateforme [DashScope](https://dashscope.console.aliyun.com/) d'Alibaba Cloud
    - Lors de votre première connexion, vous devez activer le service DashScope
    - Si vous l'avez déjà activé, vous pouvez passer cette étape

```
<img
  src="https://oss.vidol.chat/assets/297d1f46136e9fe181e980e1f6cb4019.webp"
  alt="Activer le service DashScope"
/>
```

  </Step>

  <Step title="Obtenir la clé API DashScope">
    - Accédez à l'interface `API-KEY` et créez une clé API

```
<img
  src="https://oss.vidol.chat/assets/2bdda9c970582f6ea4dfe30a66fef60d.webp"
  alt="Créer une clé API Tongyi Qianwen"
/>

- Dans la boîte de dialogue qui s'ouvre, copiez la clé API et conservez-la en lieu sûr

<img
  src="https://oss.vidol.chat/assets/8b2e4629c9fa2b75d08b00de30490b59.webp"
  alt="Copier la clé API Tongyi Qianwen"
/>

<Warning>
  Veuillez stocker la clé en toute sécurité, car elle n'apparaîtra qu'une seule fois. Si vous la perdez accidentellement, vous devrez créer une nouvelle clé.
</Warning>
```

  </Step>

  <Step title="Configurer Tongyi Qianwen dans LobeVidol">
    - Accédez à l'interface `Paramètres` de LobeVidol
    - Sous `Modèle de langage`, trouvez l'option de configuration pour `Tongyi Qianwen`

```
<img
  src="https://oss.vidol.chat/assets/ac3a3b2102f961ccc5b4d081a056cde3.webp"
  alt="Remplir la clé API"
/>

- Activez Tongyi Qianwen et saisissez la clé API obtenue
- Choisissez un modèle Qwen pour votre assistant AI afin de commencer la conversation

<img
  src="https://oss.vidol.chat/assets/16e02355939ef75e9dc361050f2b7d5b.webp"
  alt="Choisir le modèle Qwen et commencer la conversation"
/>

<Warning>
  Pendant l'utilisation, vous pourriez avoir à payer le fournisseur de services API, veuillez consulter la politique tarifaire de Tongyi Qianwen.
</Warning>
```

  </Step>
</Steps>

Vous pouvez maintenant utiliser le modèle fourni par Tongyi Qianwen pour converser dans LobeVidol.
