# 在 LobeVidol中使用 Moonshot AI

<img
 
  src="https://oss.vidol.chat/docs/2024/12/ef26c941f6d2b1d2617c7f6f93702569.png"
  alt="在 LobeVidol中使用 Moonshot AI"
/>

Moonshot AI API 现在可供所有人使用, 本文档将指导你如何在 LobeVidol中使用 [Moonshot AI](https://www.moonshot.cn/):

## 配置步骤

### 步骤一：获取 Moonshot AI API 密钥

1. 申请您的 [API 密钥](https://platform.moonshot.cn/console/api-keys)

<img
 
  src="https://oss.vidol.chat/assets/039f46519d8d785308768f01c190aed5.webp"
  alt="申请 API Key"
/>

### 步骤二：在 LobeVidol中配置 Moonshot AI

1. 访问 LobeVidol的`设置`界面
2. 在`语言模型`下找到`Moonshot AI`的设置项

<img
 
  src="https://oss.vidol.chat/assets/46bfac50760257d60be12712a7a33083.webp"
  alt="填入 API 密钥"
/>

<Warning>
如果您使用的是 mistral.ai，则您的帐户必须具有有效的订阅才能使 API 密钥正常工作。新创建的 API 密钥需要 2-3 分钟才能开始工作。如果单击"测试"按钮但失败，请在 2-3 分钟后重试。
</Warning>

3. 填入获得的 API 密钥
4. 为你的 AI 助手选择一个 Moonshot AI 的模型即可开始对话

<img
 
  src="https://oss.vidol.chat/assets/5f412973b01225a1cbd3ae3b8f14aeef.webp"
  alt="选择 Moonshot AI 模型并开始对话"
/>

<Warning>
在使用过程中你可能需要向 API 服务提供商付费，请参考 Moonshot AI 的相关费用政策。
</Warning>

至此你已经可以在 LobeVidol中使用 Moonshot AI 提供的模型进行对话了。
