# 在 LobeVidol中使用文心千帆

<Frame>
  <img
    src="https://oss.vidol.chat/docs/2024/12/9002ef959ed2146d4601aa733087253e.png"
    alt="文心千帆封面"
  />
</Frame>

[文心千帆](https://qianfan.cloud.baidu.com/)是百度推出的一个人工智能大语言模型平台，支持多种应用场景，包括文学创作、商业文案生成、数理逻辑推算等。该平台具备跨模态、跨语言的深度语义理解与生成能力，广泛应用于搜索问答、内容创作和智能办公等领域。

本文将指导你如何在 LobeVidol中使用文心千帆。

## 配置步骤

### 步骤一：获得文心千帆的 API Key

1. 注册并登录 [百度智能云控制台](https://console.bce.baidu.com/)
2. 进入 `百度智能云千帆 ModelBuilder`
3. 在左侧菜单中选择`应用接入`
4. 创建一个应用

<Frame>
  <img
    src="https://oss.vidol.chat/assets/f023683fe2f521e092cbdbebeb1de688.webp"
    alt="创建应用"
  />
</Frame>

5. 在用户账号菜单进入 `安全认证` -> `Access Key` 管理页面
6. 复制 `Access Key` 和 `Secret Key`，并妥善保存

<Frame>
  <img
    src="https://oss.vidol.chat/assets/2be6128378b48f17cd5acbe0f84dce77.webp"
    alt="保存密钥"
  />
</Frame>

### 步骤二：在 LobeVidol中配置文心千帆

1. 访问 LobeVidol的`设置`界面
2. 在`语言模型`下找到 `文心千帆` 的设置项

<Frame>
  <img
    src="https://oss.vidol.chat/assets/831813e913d9a8f9e40c5562fe50d245.webp"
    alt="填入 API 密钥"
  />
</Frame>

3. 填入获得的 `Access Key` 和 `Secret Key`
4. 为你的 AI 助手选择一个文心千帆的模型即可开始对话

<Frame>
  <img
    src="https://oss.vidol.chat/assets/768ec6fd300785186b202437985857c4.webp"
    alt="选择文心千帆模型并开始对话"
  />
</Frame>

<Warning>
  在使用过程中你可能需要向 API 服务提供商付费，请参考文心千帆的相关费用政策。
</Warning>

至此你已经可以在 LobeVidol中使用文心千帆提供的模型进行对话了。
