[360智脑](https://ai.360.com/)是360公司自主研发的认知型通用大模型，旨在为企业和开发者提供强大的自然语言处理能力。该模型已升级至4.0版本，能够支持多种应用场景，包括对话服务、图片生成、向量数据库服务等。

本文将指导你如何在 LobeVidol中使用360智脑。

## 步骤一：获得360智脑的 API Key

1. 注册并登录 [360智脑 API 开放平台](https://ai.360.com/platform/keys)
2. 点击左侧 `API Keys` 菜单
3. 创建一个 API 密钥并复制

<img
  src="https://oss.vidol.chat/assets/688e6e10904ad46cf7f44bba6359f90c.webp"
  alt="创建API密钥"
/>

## 步骤二：在 LobeVidol中配置360智脑

1. 访问 LobeVidol的`设置`界面
2. 在`语言模型`下找到 `360` 的设置项

<img
  src="https://oss.vidol.chat/assets/65c86d6e63ddd5dd9896a6a67c054c0d.webp"
  alt="填入 API 密钥"
/>

3. 填入获得的 API 密钥
4. 为你的 AI 助手选择一个360智脑的模型即可开始对话

<img
  src="https://oss.vidol.chat/docs/2024/12/e32889a1d501a14cba082ea42c7431f4.png"
  alt="选择360模型并开始对话"
/>

<Warning>在使用过程中你可能需要向 API 服务提供商付费，请参考360智脑的相关费用政策。</Warning>

至此你已经可以在 LobeVidol中使用360智脑提供的模型进行对话了。
