## Configuration Steps

### 1. Obtain a Perplexity AI API Key

1. Create an account on [Perplexity AI](https://www.perplexity.ai/)
2. Obtain your [API key](https://www.perplexity.ai/settings/api)

<img

src="https://oss.vidol.chat/assets/a34dbd6f2ee03623ea63553fb0e499b4.webp>"
alt="Create API Key"
/>

### 2. Configure Perplexity AI in LobeVidol

1. Go to the `Settings` interface in LobeVidol
2. Find the settings for `Perplexity AI` under `Language Model`

<img

src="https://oss.vidol.chat/assets/68aa02e755b6cdc605ced5507694843e.webp>"
alt="Enter API Key"
/>

3. Enter the obtained API key
4. Choose a Perplexity AI model for your AI assistant to start the conversation

<img

src="https://oss.vidol.chat/assets/a4dee38f67669435e2a2d9041800151a.webp>"
alt="Select Perplexity AI model and start conversation"
/>

<Warning>
  You may need to pay the API service provider during use, please refer to Perplexity AI's relevant pricing policy.
</Warning>

You can now use the models provided by Perplexity AI for conversations in LobeVidol.
