[Stepfun 阶跃星辰](https://www.stepfun.com/)是一家专注于通用人工智能(AGI)研发的创业公司，目前已推出Step-1千亿参数语言大模型、Step-1V千亿参数多模态大模型,以及Step-2万亿参数MoE语言大模型预览版。

本文档将指导你如何在 LobeVidol中使用 Stepfun 阶跃星辰:

## 步骤一：获取 Stepfun 阶跃星辰 API 密钥

1. 访问并登录 [Stepfun Stepfun 阶跃星辰开放平台](https://platform.stepfun.com/)
2. 进入`接口密钥`菜单，系统已为你创建好 API 密钥
3. 复制已创建的 API 密钥

<img
 
  src="https://oss.vidol.chat/assets/e15f56242cc4402902c33ac152ad0abd.webp"
  alt="获取 Stepfun 阶跃星辰API密钥"
/>

## 步骤二：在LobeVidol中配置 Stepfun Stepfun 阶跃星辰

1. 访问 LobeVidol的`设置`界面
2. 在`语言模型`下找到` Stepfun 阶跃星辰`的设置项

<img
 
  src="https://oss.vidol.chat/assets/95ffcabb47ddb989f8cb6e6d2ef451ad.webp"
  alt="填写 API 密钥"
/>

3. 打开 Stepfun 阶跃星辰并填入获得的 API 密钥
4. 为你的 AI 助手选择一个 Stepfun 阶跃星辰的模型即可开始对话

<img
 
  src="https://oss.vidol.chat/assets/771c3bd6028a014f3e0d3df01e4e4988.webp"
  alt="选择 Stepfun 阶跃星辰模型并开始对话"
/>

<Warning>在使用过程中你可能需要向 API 服务提供商付费，请参考 Stepfun 阶跃星辰的相关费用政策。</Warning>

至此你已经可以在 LobeVidol中使用 Stepfun 阶跃星辰提供的模型进行对话了。
