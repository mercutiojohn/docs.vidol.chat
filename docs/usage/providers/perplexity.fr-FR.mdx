<Note>

</Note>

## Étapes de configuration

### 1. Obtenir une clé API Perplexity AI

1. Créez un compte sur [Perplexity AI](https://www.perplexity.ai/)
2. Obtenez votre [clé API](https://www.perplexity.ai/settings/api)

<img

src="https://oss.vidol.chat/assets/a34dbd6f2ee03623ea63553fb0e499b4.webp>"
alt="Créer une clé API"
/>

### 2. Configurer Perplexity AI dans LobeVidol

1. Accédez à l'interface de `paramètres` de LobeVidol
2. Trouvez l'option de configuration `Perplexity AI` sous `modèle de langage`

<img

src="https://oss.vidol.chat/assets/68aa02e755b6cdc605ced5507694843e.webp>"
alt="Entrer la clé API"
/>

3. Entrez la clé API obtenue
4. Choisissez un modèle Perplexity AI pour votre assistant AI afin de commencer la conversation

<img

src="https://oss.vidol.chat/assets/a4dee38f67669435e2a2d9041800151a.webp>"
alt="Choisir un modèle Perplexity AI et commencer la conversation"
/>

<Warning>
  Pendant l'utilisation, vous pourriez avoir besoin de payer le fournisseur de services API, veuillez consulter la politique tarifaire de Perplexity AI.
</Warning>

Vous pouvez maintenant utiliser les modèles fournis par Perplexity AI pour converser dans LobeVidol.
