# 在 LobeVidol中使用 DeepSeek

<img
  src="https://oss.vidol.chat/docs/2024/12/58a05d76bb89eeb59e238998b69c7059.png"
  alt="在 LobeVidol中使用 DeepSeek"
/>

[DeepSeek](https://www.deepseek.com/) 是一款先进的开源大型语言模型（LLM）。最新版本 DeepSeek-V2 在架构和性能上进行了显著优化，同时训练成本降低了42.5%,推理成本降低了93.3%。

本文档将指导你如何在 LobeVidol中使用 DeepSeek:

## 步骤一：获取 DeepSeek API 密钥

1. 首先，你需要注册并登录 [DeepSeek](https://platform.deepseek.com/) 开放平台

<Info>当前新用户将会获赠 500M Tokens 的免费额度</Info>

2. 进入 `API keys` 菜单，并点击 `创建 API Key`

<img
  src="https://oss.vidol.chat/assets/498856598fdba60af6593408142db837.webp"
  alt="创建 Deepseek API 密钥"
/>

3. 在弹出的对话框中输入 API 密钥名称

<img
  src="https://oss.vidol.chat/assets/89820732dcb092627c07d24143a37f60.webp"
  alt="填写 Deepseek API 名称"
/>

4. 复制得到的 API 密钥并妥善保存

<img
  src="https://oss.vidol.chat/assets/9f3d6287b883576a6b9cf67e28a6f43e.webp"
  alt="保存 Deepseek API 密钥"
/>

<Warning>
  请安全地存储密钥，因为它只会出现一次。如果你意外丢失它，您将需要创建一个新密钥。
</Warning>

## 步骤二：在 LobeVidol中配置 DeepSeek

1. 访问 LobeVidol的 `应用设置`界面
2. 在 `语言模型` 下找到 `DeepSeek` 的设置项

<img
  src="https://oss.vidol.chat/assets/ee9802f91be4d6f1b686825b2433d75d.webp"
  alt="填写 Deepseek API 密钥"
/>

3. 打开 DeepSeek 并填入获取的 API 密钥
4. 为你的助手选择一个 DeepSeek 模型即可开始对话

<img
  src="https://oss.vidol.chat/assets/e84d84f68b6d6a8b3262dbcef2eded28.webp"
  alt="选择 Deepseek 模型"
/>

<Warning>
  在使用过程中你可能需要向 API 服务提供商付费，请参考 DeepSeek 的相关费用政策。
</Warning>

至此你已经可以在 LobeVidol中使用 Deepseek 提供的模型进行对话了。
