# 在 LobeVidol中使用百川

<img
  src="https://oss.vidol.chat/docs/2024/12/88be132f5cc8c1cfb974545d11f7897f.png"
  alt="在 LobeVidol中使用百川"
/>

本文将指导你如何在 LobeVidol中使用百川：

<Steps>
  <Step title="获取百川智能 API 密钥">
    - 创建一个[百川智能](https://platform.baichuan-ai.com/homePage)账户
    - 创建并获取 [API 密钥](https://platform.baichuan-ai.com/console/apikey)

    <img
      src="https://oss.vidol.chat/assets/217222e643d99ab3ba01fe92906f3314.webp"
      alt="创建 API Key"
    />
  </Step>

  <Step title="在 LobeVidol中配置百川">
    - 访问 LobeVidol的`设置`界面
    - 在`语言模型`下找到`百川`的设置项

    <img
      src="https://oss.vidol.chat/assets/8d4fbb776e2209a1ec58c6b3516351a1.webp"
      alt="填入 API 密钥"
    />

    - 填入获得的 API 密钥
    - 为你的 AI 助手选择一个百川的模型即可开始对话

    <img
      
      src="https://oss.vidol.chat/assets/2cfe64ead120815f7ba7100bc3dcfd48.webp"
      alt="选择百川模型并开始对话"
    />
  </Step>
</Steps>

<Warning>在使用过程中你可能需要向 API 服务提供商付费，请参考百川的相关费用政策。</Warning>

至此你已经可以在 LobeVidol中使用百川提供的模型进行对话了。
