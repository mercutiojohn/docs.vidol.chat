## Étapes de configuration

### 1. Obtenir une clé API OpenAI

Pour utiliser les services d'OpenAI, vous devez :

1. Créer un compte OpenAI :
   - Visitez la [page d'inscription d'OpenAI](https://platform.openai.com/signup)
   - Utilisez un numéro de téléphone international (hors de la Chine continentale)
   - Utilisez une adresse e-mail en dehors de la Chine continentale
2. Créer une clé API :
   - Après vous être connecté, accédez à la [page des clés API](https://platform.openai.com/api-keys)
   - Cliquez sur le bouton `Créer une nouvelle clé secrète`

</Warning>

### Services de proxy recommandés

Si vous rencontrez des difficultés pour accéder aux services d'OpenAI, envisagez d'utiliser des services de proxy, tels que [OpenRouter](https://openrouter.ai/) ou [AIHubMix](https://aihubmix.com?aff=NBh9) comme fournisseur de services de proxy fiable.

Les services de proxy offrent les avantages suivants :

- Accès API stable
- Tarifs raisonnables
- Méthodes de recharge pratiques
- Support client de qualité

Utiliser un service de proxy peut vous aider à utiliser les services d'OpenAI plus facilement, en particulier pour les utilisateurs situés dans certaines régions géographiques.

<Warning>
  Veuillez noter qu'en utilisant des services de proxy tiers, vous devez vous assurer de respecter les lois et règlements en vigueur et de protéger la sécurité de vos informations personnelles.
</Warning>

### 2. Configurer OpenAI dans LobeVidol

1. Ouvrir les paramètres :
   - Accédez à l'interface `Paramètres` de LobeVidol
   - Trouvez la section `Modèle de langage`
   - Sélectionnez l'option de configuration `OpenAI`

<Frame>
  <img 
    src="https://oss.vidol.chat/docs/2024/11/65be98b535abfdf1de172f90f1f4d087.png"
    alt="Entrer la clé API"
  />
</Frame>

2. Configurer le service :
   - Entrez votre clé API obtenue dans le champ correspondant
   - Sélectionnez le modèle OpenAI que vous souhaitez utiliser
   - Optionnel : configurez une adresse de proxy API personnalisée (si nécessaire)

  <img 
    src="https://oss.vidol.chat/docs/2024/11/337b8df7091c4a261979de943d1474cf.png"

  />
</Frame>

<Warning>
  L'utilisation des services OpenAI nécessite un paiement. Les tarifs varient selon les modèles, pour plus de détails, veuillez consulter [la page de tarification d'OpenAI](https://openai.com/pricing).
</Warning>

## Questions fréquentes

<Accordion title="La clé API ne fonctionne pas ?">
  - Vérifiez si la clé API a été correctement copiée
  - Vérifiez si le solde de votre compte est suffisant
  - Assurez-vous que vous êtes dans une région où les services OpenAI sont disponibles
</Accordion>

<Accordion title="Comment choisir le modèle approprié ?">
  - Série GPT-4 : performances supérieures, adaptée aux tâches complexes
  - Série GPT-3.5 : bon rapport qualité-prix, adaptée aux conversations générales
</Accordion>

<Accordion title="Questions de facturation">
  - Facturation basée sur l'utilisation réelle
  - Tarifs différents selon les modèles
  - Il est conseillé de vérifier régulièrement votre utilisation
</Accordion>
