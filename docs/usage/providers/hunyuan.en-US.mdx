## Overview

[Tencent Hunyuan](https://hunyuan.tencent.com/) is a large model launched by Tencent, designed to provide users with intelligent assistant services. It utilizes natural language processing technology to help users solve problems, offer suggestions, and generate content. Users can quickly obtain the information they need through conversations with the model, thereby improving work efficiency.

This article will guide you on how to use Tencent Hunyuan in LobeVidol.

## Configuration Steps

### 1. Obtain Tencent Hunyuan API Key

1. Register and log in to the [Tencent Cloud Console](https://console.cloud.tencent.com/hunyuan/api-key)
2. Navigate to `Hunyuan Large Model` and click on `API KEY Management`
3. Create an API key

<img
src="https://oss.vidol.chat/assets/01dbe69cc092163f7ac782afb7d314c5.webp"
alt="Create API Key"
/>

4. Click `View`, and copy the API key from the pop-up panel, ensuring you save it securely

<img
src="https://oss.vidol.chat/assets/856bd407c8a1510f616a4bdb1e02a883.webp"
alt="Save Key"
/>

### 2. Configure Tencent Hunyuan in LobeVidol

1. Access the `Settings` interface in LobeVidol
2. Find the `Tencent Hunyuan` settings under `Language Model`

<img
src="https://oss.vidol.chat/assets/bd8c97ef67055e3ff93c56e46c33fa8d.webp"
alt="Enter API Key"
/>

3. Enter the obtained API key
4. Choose a Tencent Hunyuan model for your AI assistant to start the conversation

<img
src="https://oss.vidol.chat/assets/3c54d6f2d55fae843fbbfdc0bd7ffec7.webp"
alt="Select Tencent Hunyuan Model and Start Conversation"
/>

<Warning>
  You may need to pay the API service provider during use; please refer to Tencent Hunyuan's relevant pricing policy.
</Warning>

You can now use the models provided by Tencent Hunyuan for conversations in LobeVidol.
