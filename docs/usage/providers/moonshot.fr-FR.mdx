# Utiliser Moonshot AI dans LobeVidol

<img

src="https://oss.vidol.chat/docs/2024/12/ef26c941f6d2b1d2617c7f6f93702569.png"
alt="Utiliser Moonshot AI dans LobeVidol"
/>

L'API Moonshot AI est maintenant disponible pour tous. Ce document vous guidera sur la façon d'utiliser [Moonshot AI](https://www.moonshot.cn/) dans LobeVidol :

## Étapes de configuration

### Étape 1 : Obtenir une clé API Moonshot AI

1. Demandez votre [clé API](https://platform.moonshot.cn/console/api-keys)

<img

src="https://oss.vidol.chat/assets/039f46519d8d785308768f01c190aed5.webp>"
alt="Demander une clé API"
/>

### Étape 2 : Configurer Moonshot AI dans LobeVidol

1. Accédez à l'interface de `paramètres` de LobeVidol
2. Sous `modèle de langage`, trouvez l'option de configuration pour `Moonshot AI`

<img

src="https://oss.vidol.chat/assets/46bfac50760257d60be12712a7a33083.webp>"
alt="Entrer la clé API"
/>

<Warning>
Si vous utilisez mistral.ai, votre compte doit avoir un abonnement valide pour que la clé API fonctionne correctement. Les nouvelles clés API peuvent prendre 2 à 3 minutes pour commencer à fonctionner. Si vous cliquez sur le bouton "Tester" mais que cela échoue, veuillez réessayer dans 2 à 3 minutes.
</Warning>

3. Entrez la clé API obtenue
4. Choisissez un modèle Moonshot AI pour votre assistant AI afin de commencer la conversation

<img

src="https://oss.vidol.chat/assets/5f412973b01225a1cbd3ae3b8f14aeef.webp>"
alt="Choisir un modèle Moonshot AI et commencer la conversation"
/>

<Warning>
Au cours de l'utilisation, vous devrez peut-être payer le fournisseur de services API, veuillez consulter la politique tarifaire de Moonshot AI.
</Warning>

Vous pouvez maintenant utiliser les modèles fournis par Moonshot AI pour converser dans LobeVidol.
