## 概述

[腾讯混元](https://hunyuan.tencent.com/)是由腾讯推出的一款大模型，旨在为用户提供智能助手服务。它能够通过自然语言处理技术，帮助用户解决问题、提供建议以及进行内容生成等任务。用户可以通过与模型的对话，快速获取所需信息，从而提高工作效率。

本文将指导你如何在 LobeVidol中使用腾讯混元。

## 配置步骤

### 1. 获得腾讯混元的 API Key

1. 注册并登录 [腾讯云控制台](https://console.cloud.tencent.com/hunyuan/api-key)
2. 进入 `混元大模型` 并点击 `API KEY 管理`
3. 创建一个 API 密钥

<img
  src="https://oss.vidol.chat/assets/01dbe69cc092163f7ac782afb7d314c5.webp"
  alt="创建 API 密钥"
/>

4. 点击`查看`，在弹出面板中复制 API 密钥，并妥善保存

<img
  src="https://oss.vidol.chat/assets/856bd407c8a1510f616a4bdb1e02a883.webp"
  alt="保存密钥"
/>

### 2. 在 LobeVidol中配置腾讯混元

1. 访问 LobeVidol的`设置`界面
2. 在`语言模型`下找到 `腾讯混元` 的设置项

<img
  src="https://oss.vidol.chat/assets/bd8c97ef67055e3ff93c56e46c33fa8d.webp"
  alt="填入 API 密钥"
/>

3. 填入获得的 API 密钥
4. 为你的 AI 助手选择一个腾讯混元的模型即可开始对话

<img
  src="https://oss.vidol.chat/assets/3c54d6f2d55fae843fbbfdc0bd7ffec7.webp"
  alt="选择腾讯混元模型并开始对话"
/>

<Warning>
  在使用过程中你可能需要向 API 服务提供商付费，请参考腾讯混元的相关费用政策。
</Warning>

至此你已经可以在 LobeVidol中使用腾讯混元提供的模型进行对话了。
