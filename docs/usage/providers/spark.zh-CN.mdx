[讯飞星火](https://xinghuo.xfyun.cn/)是科大讯飞推出的一款强大的AI大模型，具备跨领域的知识和语言理解能力，能够执行问答、对话和文学创作等多种任务。

本文将指导你如何在 LobeVidol中使用讯飞星火。

## 步骤一：获得讯飞星火的 API Key

1. 注册并登录 [讯飞开放平台](https://console.xfyun.cn/)
2. 创建一个应用

<img
 
  src="https://oss.vidol.chat/assets/4d0d191b487c114abf084eb7f2dc381c.webp"
  alt="创建应用"
/>

3. 选择一个大模型查看详情
4. 复制右上角 http 服务接口认证信息中的 `API Password`

<img
 
  src="https://oss.vidol.chat/assets/8669131e67e5276fe0744754ba4b1645.webp"
  alt="复制 API 密钥"
/>

## 步骤二：在 LobeVidol中配置讯飞星火

1. 访问 LobeVidol的`设置`界面
2. 在`语言模型`下找到 `讯飞星火` 的设置项

<img
 
  src="https://oss.vidol.chat/assets/4169b5d9f7534f9f89c8426445e9a080.webp"
  alt="填入 API 密钥"
/>

3. 填入获得的 API 密钥
4. 为你的 AI 助手选择一个讯飞星火的模型即可开始对话

<img
 
  src="https://oss.vidol.chat/assets/4175fc55c2093d635f15a3287e89e977.webp"
  alt="选择讯飞星火模型并开始对话"
/>

<Warning>在使用过程中你可能需要向 API 服务提供商付费，请参考讯飞星火的相关费用政策。</Warning>

至此你已经可以在 LobeVidol中使用讯飞星火提供的模型进行对话了。
