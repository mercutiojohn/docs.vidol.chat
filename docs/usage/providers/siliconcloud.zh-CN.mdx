## 概述

[SiliconCloud](https://siliconflow.cn/) 是一个基于开源基础模型的人工智能服务平台，提供多种生成式AI（GenAI）服务。

本文将指导你如何在 LobeVidol中使用 SiliconCloud。

<img
 
  src="https://oss.vidol.chat/docs/2024/12/f3fc8b92a6bbfa136f2f178afb830700.png"
  alt="SiliconCloud Banner"
/>

## 配置步骤

### 1. 获得 SiliconCloud 的 API Key

1. 注册并登录 [SiliconCloud](https://cloud.siliconflow.cn/account/ak)
2. 点击左侧 `API 密钥` 菜单
3. 创建一个 API 密钥并复制

<img
  src="https://oss.vidol.chat/assets/56e5331e7fae3754820790c824cdc480.webp"
  alt="创建API密钥"
/>

### 2. 在 LobeVidol中配置 SiliconCloud 

1. 访问 LobeVidol的`设置`界面
2. 在`语言模型`下找到 `SiliconFlow` 的设置项
3. 填入获得的 API 密钥

<img
  src="https://oss.vidol.chat/assets/94f55c97a24a08c7a5923c23ee2d7eef.webp"
  alt="填入 API 密钥"
/>

### 3. 开始使用

为你的 AI 助手选择一个 SiliconCloud 的模型即可开始对话

<img
  src="https://oss.vidol.chat/assets/94397c91265c37b9f313dc439b90125f.webp"
  alt="选择 SiliconCloud 模型并开始对话"
/>

<Warning>
  在使用过程中你可能需要向 API 服务提供商付费，请参考 SiliconCloud 的相关费用政策。
</Warning>

至此你已经可以在 LobeVidol中使用 SiliconCloud 提供的模型进行对话了。
