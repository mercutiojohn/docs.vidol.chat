# Utiliser DeepSeek dans LobeVidol

<img
src="https://oss.vidol.chat/docs/2024/12/58a05d76bb89eeb59e238998b69c7059.png"
alt="Utiliser DeepSeek dans LobeVidol"
/>

[DeepSeek](https://www.deepseek.com/) est un modèle de langage avancé open source (LLM). La dernière version, DeepSeek-V2, a été significativement optimisée en termes d'architecture et de performance, avec une réduction des coûts d'entraînement de 42,5 % et des coûts d'inférence de 93,3 %.

Ce document vous guidera sur la façon d'utiliser DeepSeek dans LobeVidol :

## Étape 1 : Obtenir une clé API DeepSeek

1. Tout d'abord, vous devez vous inscrire et vous connecter à la plateforme ouverte [DeepSeek](https://platform.deepseek.com/)

<Info>Les nouveaux utilisateurs recevront actuellement un crédit gratuit de 500M Tokens</Info>

2. Accédez au menu `Clés API` et cliquez sur `Créer une clé API`

<img
src="https://oss.vidol.chat/assets/498856598fdba60af6593408142db837.webp"
alt="Créer une clé API DeepSeek"
/>

3. Dans la boîte de dialogue qui s'ouvre, saisissez le nom de la clé API

<img
src="https://oss.vidol.chat/assets/89820732dcb092627c07d24143a37f60.webp"
alt="Saisir le nom de la clé API DeepSeek"
/>

4. Copiez la clé API obtenue et conservez-la en lieu sûr

<img
src="https://oss.vidol.chat/assets/9f3d6287b883576a6b9cf67e28a6f43e.webp"
alt="Conserver la clé API DeepSeek"
/>

<Warning>
  Veuillez stocker la clé en toute sécurité, car elle n'apparaîtra qu'une seule fois. Si vous la perdez accidentellement, vous devrez créer une nouvelle clé.
</Warning>

## Étape 2 : Configurer DeepSeek dans LobeVidol

1. Accédez à l'interface `Paramètres de l'application` de LobeVidol
2. Sous `Modèle de langage`, trouvez les paramètres de `DeepSeek`

<img
src="https://oss.vidol.chat/assets/ee9802f91be4d6f1b686825b2433d75d.webp"
alt="Saisir la clé API DeepSeek"
/>

3. Activez DeepSeek et saisissez la clé API obtenue
4. Choisissez un modèle DeepSeek pour votre assistant afin de commencer à dialoguer

<img
src="https://oss.vidol.chat/assets/e84d84f68b6d6a8b3262dbcef2eded28.webp"
alt="Choisir un modèle DeepSeek"
/>

<Warning>
  Pendant l'utilisation, vous devrez peut-être payer le fournisseur de services API, veuillez consulter la politique tarifaire de DeepSeek.
</Warning>

Vous pouvez maintenant utiliser les modèles fournis par DeepSeek pour dialoguer dans LobeVidol.
