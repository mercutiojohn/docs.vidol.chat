[Fireworks.ai](https://fireworks.ai/) 是一个高性能的生成式 AI 模型推理平台，允许用户通过其 API 访问和使用各种模型。该平台支持多种模态，包括文本和视觉语言模型，并提供函数调用和 JSON 模式等功能，以增强应用开发的灵活性。

本文将指导你如何在 LobeVidol中使用 Fireworks AI。

## 配置步骤

### 1. 获得 Fireworks AI 的 API Key

1. 登录 [Fireworks.ai 控制台](https://fireworks.ai/account/api-keys)
2. 进入 `User` 页面，点击 `API Keys`
3. 创建一个新的 API 密钥

<img
  src="https://oss.vidol.chat/assets/d0c4369f894abb5ad6e514059b8f378e.webp"
  alt="创建 API 密钥"
/>

4. 复制并保存生成的 API 密钥

<img
  src="https://oss.vidol.chat/assets/89a8dadc85902334ce8d2d5b78abf709.webp"
  alt="保存 API 密钥"
/>

<Warning>
  请安全地存储密钥，因为它只会出现一次。如果您意外丢失它，您将需要创建一个新密钥。
</Warning>

### 2. 在 LobeVidol中配置 Fireworks AI

1. 访问 LobeVidol的`设置`界面
2. 在`语言模型`下找到 `Fireworks AI` 的设置项

<img
  src="https://oss.vidol.chat/assets/74f5de8ee68e57b472d1fc56a2df40d5.webp"
  alt="填入 API 密钥"
/>

3. 填入获得的 API 密钥
4. 为你的 AI 助手选择一个 Fireworks AI 的模型即可开始对话

<img
  src="https://oss.vidol.chat/assets/ba0243e75b0421b6dd7dadad02e4b0d6.webp"
  alt="选择 Fireworks AI 模型并开始对话"
/>

<Warning>
  在使用过程中你可能需要向 API 服务提供商付费，请参考 Fireworks AI 的相关费用政策。
</Warning>

至此你已经可以在 LobeVidol中使用 Fireworks AI 提供的模型进行对话了。
