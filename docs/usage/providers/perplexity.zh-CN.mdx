</Note>

## 配置步骤

### 1. 获取 Perplexity AI API 密钥

1. 创建一个 [Perplexity AI](https://www.perplexity.ai/) 帐户
2. 获取您的 [API 密钥](https://www.perplexity.ai/settings/api)

<img
 
  src="https://oss.vidol.chat/assets/a34dbd6f2ee03623ea63553fb0e499b4.webp"
  alt="创建 API Key"
/>

### 2. 在 LobeVidol中配置 Perplexity AI

1. 访问 LobeVidol的`设置`界面
2. 在`语言模型`下找到`Perplexity AI`的设置项

<img
 
  src="https://oss.vidol.chat/assets/68aa02e755b6cdc605ced5507694843e.webp"
  alt="填入 API 密钥"
/>

3. 填入获得的 API 密钥
4. 为你的 AI 助手选择一个 Perplexity AI 的模型即可开始对话

<img
 
  src="https://oss.vidol.chat/assets/a4dee38f67669435e2a2d9041800151a.webp"
  alt="选择 Perplexity AI 模型并开始对话"
/>

<Warning>
  在使用过程中你可能需要向 API 服务提供商付费，请参考 Perplexity AI 的相关费用政策。
</Warning>

至此你已经可以在 LobeVidol中使用 Perplexity AI 提供的模型进行对话了。
