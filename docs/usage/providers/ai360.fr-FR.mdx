[360 Zhinao](https://ai.360.com/) est un modèle général cognitif développé par la société 360, visant à fournir aux entreprises et aux développeurs de puissantes capacités de traitement du langage naturel. Ce modèle a été mis à jour vers la version 4.0, capable de prendre en charge divers scénarios d'application, y compris les services de dialogue, la génération d'images, et les services de bases de données vectorielles.

Cet article vous guidera sur la façon d'utiliser 360 Zhinao dans LobeVidol.

## Étape 1 : Obtenir la clé API de 360 Zhinao

1. Inscrivez-vous et connectez-vous à la [plateforme ouverte API de 360 Zhinao](https://ai.360.com/platform/keys)
2. Cliquez sur le menu `Clés API` à gauche
3. Créez une clé API et copiez-la

<img
src="https://oss.vidol.chat/assets/688e6e10904ad46cf7f44bba6359f90c.webp"
alt="Créer une clé API"
/>

## Étape 2 : Configurer 360 Zhinao dans LobeVidol

1. Accédez à l'interface `Paramètres` de LobeVidol
2. Sous `Modèle de langage`, trouvez l'option de configuration pour `360`

<img
src="https://oss.vidol.chat/assets/65c86d6e63ddd5dd9896a6a67c054c0d.webp"
alt="Entrer la clé API"
/>

3. Entrez la clé API obtenue
4. Choisissez un modèle de 360 Zhinao pour votre assistant AI afin de commencer le dialogue

<img
src="https://oss.vidol.chat/docs/2024/12/e32889a1d501a14cba082ea42c7431f4.png"
alt="Choisir un modèle 360 et commencer le dialogue"
/>

<Warning>Au cours de l'utilisation, vous pourriez avoir besoin de payer le fournisseur de services API, veuillez consulter la politique tarifaire de 360 Zhinao.</Warning>

Vous pouvez maintenant utiliser les modèles fournis par 360 Zhinao pour dialoguer dans LobeVidol.
