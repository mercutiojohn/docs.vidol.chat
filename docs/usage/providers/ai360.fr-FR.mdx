[360智脑](https://ai.360.com/) est un modèle général cognitif développé par la société 360, visant à fournir aux entreprises et aux développeurs de puissantes capacités de traitement du langage naturel. Ce modèle a été mis à jour vers la version 4.0, capable de prendre en charge divers scénarios d'application, y compris les services de dialogue, la génération d'images, et les services de bases de données vectorielles.

Cet article vous guidera sur la façon d'utiliser 360智脑 dans LobeChat.

## Étape 1 : Obtenir la clé API de 360智脑

1. Inscrivez-vous et connectez-vous à la [plateforme ouverte API de 360智脑](https://ai.360.com/platform/keys)
2. Cliquez sur le menu `Clés API` à gauche
3. Créez une clé API et copiez-la

<img
src="https://github.com/user-attachments/assets/72da7af1-e180-4759-84a5-a6f6ca28392e"
alt="Créer une clé API"
/>

## Étape 2 : Configurer 360智脑 dans LobeChat

1. Accédez à l'interface de `paramètres` de LobeChat
2. Sous `modèle de langage`, trouvez l'option de configuration pour `360`

<img
src="https://github.com/user-attachments/assets/a53deb11-2c14-441a-8a5c-a0f3a74e2a63"
alt="Entrer la clé API"
/>

3. Entrez la clé API obtenue
4. Choisissez un modèle de 360智脑 pour votre assistant AI afin de commencer le dialogue

<img
src="https://github.com/user-attachments/assets/452d0b48-5ff7-4f42-a46e-68a62b87632b"
alt="Choisir un modèle 360 et commencer le dialogue"
/>

<Warning>Au cours de l'utilisation, vous pourriez avoir besoin de payer le fournisseur de services API, veuillez consulter la politique tarifaire de 360智脑.</Warning>

Vous pouvez maintenant utiliser le modèle fourni par 360智脑 pour dialoguer dans LobeChat.
