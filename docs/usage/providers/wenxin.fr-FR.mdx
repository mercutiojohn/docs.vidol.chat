# Utiliser Wenxin Qianfan dans LobeVidol

<Frame>
  <img
    src="https://oss.vidol.chat/docs/2024/12/9002ef959ed2146d4601aa733087253e.png"
    alt="Couverture de Wenxin Qianfan"
  />
</Frame>

[Wenxin Qianfan](https://qianfan.cloud.baidu.com/) est une plateforme de modèle de langage AI lancée par Baidu, prenant en charge divers scénarios d'application, y compris la création littéraire, la génération de contenu commercial, et le raisonnement logique mathématique. Cette plateforme possède des capacités de compréhension et de génération sémantique profonde multimodale et multilingue, largement utilisée dans des domaines tels que les questions-réponses, la création de contenu et le bureau intelligent.

Cet article vous guidera sur la façon d'utiliser Wenxin Qianfan dans LobeVidol.

## Étapes de configuration

### Étape 1 : Obtenir la clé API de Wenxin Qianfan

1. Inscrivez-vous et connectez-vous à [la console de cloud intelligent Baidu](https://console.bce.baidu.com/)
2. Accédez à `Baidu Intelligent Cloud Qianfan ModelBuilder`
3. Dans le menu de gauche, sélectionnez `Accès à l'application`
4. Créez une application

<Frame>
  <img
    src="https://oss.vidol.chat/assets/f023683fe2f521e092cbdbebeb1de688.webp"
    alt="Créer une application"
  />
</Frame>

5. Dans le menu de votre compte utilisateur, accédez à `Authentification de sécurité` -> page de gestion `Access Key`
6. Copiez `Access Key` et `Secret Key`, et conservez-les en lieu sûr

<Frame>
  <img
    src="https://oss.vidol.chat/assets/2be6128378b48f17cd5acbe0f84dce77.webp"
    alt="Sauvegarder les clés"
  />
</Frame>

### Étape 2 : Configurer Wenxin Qianfan dans LobeVidol

1. Accédez à l'interface `Paramètres` de LobeVidol
2. Sous `Modèle de langage`, trouvez l'option de configuration pour `Wenxin Qianfan`

<Frame>
  <img
    src="https://oss.vidol.chat/assets/831813e913d9a8f9e40c5562fe50d245.webp"
    alt="Entrer la clé API"
  />
</Frame>

3. Entrez l'`Access Key` et le `Secret Key` obtenus
4. Choisissez un modèle Wenxin Qianfan pour votre assistant AI afin de commencer à dialoguer

<Frame>
  <img
    src="https://oss.vidol.chat/assets/768ec6fd300785186b202437985857c4.webp"
    alt="Choisir le modèle Wenxin Qianfan et commencer à dialoguer"
  />
</Frame>

<Warning>
  Pendant l'utilisation, vous pourriez avoir besoin de payer le fournisseur de services API, veuillez consulter la politique tarifaire de Wenxin Qianfan.
</Warning>

Vous pouvez maintenant utiliser les modèles fournis par Wenxin Qianfan pour dialoguer dans LobeVidol.
