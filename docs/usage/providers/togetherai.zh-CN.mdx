[together.ai](https://www.together.ai/) 是一家专注于生成式人工智能(AIGC)领域的平台,成立于2022年6月。 它致力于构建用于运行、训练和微调开源模型的云平台,以低于主流供应商的价格提供可扩展的计算能力。

本文档将指导你如何在 LobeVidol中使用 Together AI:

## 步骤一：获取 Together AI 的 API 密钥

1. 访问并登录 [Together AI API](https://api.together.ai/)
2. 初次登录时系统会自动为你创建好 API 密钥并赠送 $5.0 的额度

<img
 
  src="https://oss.vidol.chat/assets/963ea14c00335f0e9369901d6dba79b2.webp"
  alt="获得 together.ai API 密钥"
/>

3. 如果你没有保存，也可以在后续任意时间，通过 `设置` 中的 `API 密钥` 界面查看

<img
 
  src="https://oss.vidol.chat/assets/47538a534659ed993831615752c23552.webp"
  alt="查看 together.ai API 密钥"
/>

## 步骤二：在 LobeVidol中配置 Together AI

1. 访问LobeVidol的`设置`界面
2. 在`语言模型`下找到`together.ai`的设置项

<img
 
  src="https://oss.vidol.chat/assets/305a102705f5379f5d2b795793312ba6.webp"
  alt="LobeVidol中填写 Together AI API 密钥"
/>

3. 打开 together.ai 并填入获得的 API 密钥
4. 为你的助手选择一个 Together AI 的模型即可开始对话

<img
 
  src="https://oss.vidol.chat/assets/8ab9ae6ed6282bcdd6473b18d68a2eda.webp"
  alt="选择并使用 Together AI 模型"
/>

<Warning>在使用过程中你可能需要向 API 服务提供商付费，请参考 Together AI 的费用政策。</Warning>

至此你已经可以在 LobeVidol中使用 Together AI 提供的模型进行对话了。
