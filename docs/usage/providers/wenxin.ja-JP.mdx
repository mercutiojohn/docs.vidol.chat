# LobeVidolで文心千帆を使用する

<Frame>
  <img
    src="https://oss.vidol.chat/docs/2024/12/9002ef959ed2146d4601aa733087253e.png"
    alt="文心千帆のカバー"
  />
</Frame>

[文心千帆](https://qianfan.cloud.baidu.com/)は、百度が提供する人工知能大言語モデルプラットフォームで、文学創作、ビジネス文書生成、数理論理推論など、さまざまなアプリケーションシーンをサポートしています。このプラットフォームは、クロスモーダルおよびクロス言語の深い意味理解と生成能力を備えており、検索質問応答、コンテンツ作成、スマートオフィスなどの分野で広く利用されています。

この記事では、LobeVidolで文心千帆を使用する方法を説明します。

## 設定手順

### 手順1：文心千帆のAPIキーを取得する

1. [百度智能云コントロールパネル](https://console.bce.baidu.com/)に登録してログインします。
2. `百度智能云千帆 ModelBuilder`にアクセスします。
3. 左側のメニューから`アプリケーション接続`を選択します。
4. アプリケーションを作成します。

<Frame>
  <img
    src="https://oss.vidol.chat/assets/f023683fe2f521e092cbdbebeb1de688.webp"
    alt="アプリケーションの作成"
  />
</Frame>

5. ユーザーアカウントメニューから`セキュリティ認証` -> `Access Key`管理ページに移動します。
6. `Access Key`と`Secret Key`をコピーし、安全に保存します。

<Frame>
  <img
    src="https://oss.vidol.chat/assets/2be6128378b48f17cd5acbe0f84dce77.webp"
    alt="キーの保存"
  />
</Frame>

### 手順2：LobeVidolで文心千帆を設定する

1. LobeVidolの`設定`画面にアクセスします。
2. `言語モデル`の下で`文心千帆`の設定項目を見つけます。

<Frame>
  <img
    src="https://oss.vidol.chat/assets/831813e913d9a8f9e40c5562fe50d245.webp"
    alt="APIキーを入力"
  />
</Frame>

3. 取得した`Access Key`と`Secret Key`を入力します。
4. AIアシスタントのために文心千帆のモデルを選択し、対話を開始します。

<Frame>
  <img
    src="https://oss.vidol.chat/assets/768ec6fd300785186b202437985857c4.webp"
    alt="文心千帆モデルを選択して対話を開始"
  />
</Frame>

<Warning>
  使用中にAPIサービスプロバイダーに料金を支払う必要がある場合がありますので、文心千帆の関連料金ポリシーを参照してください。
</Warning>

これで、LobeVidolで文心千帆が提供するモデルを使用して対話を行うことができるようになりました。
