## 概要

[Tencent Hunyuan](https://hunyuan.tencent.com/)は、Tencentが提供する大規模モデルで、ユーザーにインテリジェントアシスタントサービスを提供することを目的としています。自然言語処理技術を通じて、ユーザーが問題を解決したり、提案を提供したり、コンテンツ生成などのタスクを実行するのを支援します。ユーザーはモデルとの対話を通じて、迅速に必要な情報を取得し、作業効率を向上させることができます。

この記事では、LobeVidolでTencent Hunyuanを使用する方法を説明します。

## 設定手順

### 1. Tencent HunyuanのAPIキーを取得する

1. [Tencent Cloud Console](https://console.cloud.tencent.com/hunyuan/api-key)に登録してログインします。
2. `Hunyuan大モデル`に移動し、`API KEY管理`をクリックします。
3. APIキーを作成します。

<img
src="https://oss.vidol.chat/assets/01dbe69cc092163f7ac782afb7d314c5.webp"
alt="APIキーの作成"
/>

4. `表示`をクリックし、ポップアップパネルでAPIキーをコピーして安全に保存します。

<img
src="https://oss.vidol.chat/assets/856bd407c8a1510f616a4bdb1e02a883.webp"
alt="キーの保存"
/>

### 2. LobeVidolでTencent Hunyuanを設定する

1. LobeVidolの`設定`画面にアクセスします。
2. `言語モデル`の下で`Tencent Hunyuan`の設定項目を見つけます。

<img
src="https://oss.vidol.chat/assets/bd8c97ef67055e3ff93c56e46c33fa8d.webp"
alt="APIキーを入力"
/>

3. 取得したAPIキーを入力します。
4. AIアシスタントのためにTencent Hunyuanのモデルを選択し、対話を開始します。

<img
src="https://oss.vidol.chat/assets/3c54d6f2d55fae843fbbfdc0bd7ffec7.webp"
alt="Tencent Hunyuanモデルを選択して対話を開始"
/>

<Warning>
  使用中にAPIサービスプロバイダーに料金を支払う必要がある場合がありますので、Tencent Hunyuanの関連料金ポリシーを参照してください。
</Warning>

これで、LobeVidolでTencent Hunyuanが提供するモデルを使用して対話を行うことができるようになりました。
