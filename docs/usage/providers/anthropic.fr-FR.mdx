## Étapes de configuration

### Étape 1 : Obtenir la clé API Anthropic Claude

1. Inscrivez-vous et créez un compte sur [Anthropic Claude API](https://www.anthropic.com/api)
2. Récupérez votre [clé API](https://console.anthropic.com/settings/keys) dans la console

<Frame>
  <img 
    src="https://oss.vidol.chat/assets/ec6bd7ee7851ae4d9289dc9af7f73c1c.webp"
    alt="Créer une clé API"
  />
</Frame>

<Warning>
  L'API Claude propose actuellement 5 dollars de crédits gratuits, mais cela ne s'applique qu'à certains pays/régions. Vous pouvez vérifier si cela s'applique à votre région dans le tableau de bord > Réclamer. 
  Si ce n'est pas le cas, vous pouvez envisager d'autres méthodes de paiement.
</Warning>

3. Configurez vos informations de facturation sur https://console.anthropic.com/settings/plans choisissez le plan "Génération" pour activer le mode de paiement à l'utilisation

<Frame>
  <img 
    src="https://oss.vidol.chat/assets/66b784fffe7573242d86f4f73f177ce0.webp"
    alt="Configurer votre facturation"
  />
</Frame>

### Étape 2 : Configurer Anthropic Claude dans LobeVidol

1. Ouvrez la page `Paramètres` de LobeVidol
2. Trouvez l'option de configuration `Anthropic Claude` sous la section `Paramètres du modèle AI`

<Frame>
  <img 
    src="https://oss.vidol.chat/assets/feb270dee460367739857f028f2f617e.webp"
    alt="Entrer la clé API"
  />
</Frame>

3. Entrez votre clé API
4. Choisissez une version du modèle Anthropic Claude qui correspond à vos besoins :
   - Claude 3 Opus : le modèle le plus puissant, idéal pour la génération de scripts vidéo complexes
   - Claude 3.5 Sonnet : un choix idéal équilibrant performance et coût
   - Claude 3 Haiku : le plus rapide, adapté pour des idées créatives rapides

<Frame>
  <img 
    src="https://oss.vidol.chat/assets/4f9af602892219fa812b4321b1b2cdcd.webp"
    alt="Choisir un modèle Anthropic Claude et commencer à créer"
  />
</Frame>

<Warning>
  L'utilisation des services API nécessite le paiement de frais correspondants. Pour les tarifs spécifiques, veuillez consulter la politique de prix officielle d'Anthropic Claude. Il est conseillé de bien comprendre les critères de facturation de chaque modèle avant utilisation.
</Warning>

### Étape 3 : Commencer à utiliser

Une fois la configuration terminée, vous pouvez tirer parti des puissantes fonctionnalités d'Anthropic Claude pour :

- Générer des scripts vidéo créatifs
- Optimiser les titres et descriptions de vidéos
- Obtenir des suggestions de montage intelligentes
- Générer des sous-titres et des textes de voix off
- Créer du contenu multilingue

## Questions Fréquemment Posées

<AccordionGroup>
  <Accordion title="Comment choisir le modèle Claude approprié ?">
    - Claude 3 Opus : adapté aux projets nécessitant une réflexion approfondie et des idées complexes
    - Claude 3.5 Sonnet : le meilleur choix pour la création quotidienne
    - Claude 3 Haiku : idéal pour des itérations rapides et des tâches simples
  </Accordion>

  <Accordion title="Que faire si la clé API ne fonctionne pas ?">
    1. Vérifiez si les informations de facturation sont correctement configurées
    2. Assurez-vous que la clé API est activée
    3. Vérifiez que le solde de votre compte est suffisant
    4. Vérifiez s'il existe des restrictions régionales
  </Accordion>

  <Accordion title="Comment optimiser les coûts d'utilisation de l'API ?">
    1. Choisissez judicieusement la version du modèle
    2. Utilisez des mots-clés plus précis
    3. Traitez en lot des tâches similaires
    4. Surveillez régulièrement votre consommation
  </Accordion>
