[iFlytek Xinghuo](https://xinghuo.xfyun.cn/) is a powerful AI large model launched by iFlytek, equipped with cross-domain knowledge and language understanding capabilities, capable of performing various tasks such as Q&A, dialogue, and literary creation.

This article will guide you on how to use iFlytek Xinghuo in LobeVidol.

## Step 1: Obtain the iFlytek Xinghuo API Key

1. Register and log in to the [iFlytek Open Platform](https://console.xfyun.cn/)
2. Create an application

<img

src="https://oss.vidol.chat/assets/4d0d191b487c114abf084eb7f2dc381c.webp>"
alt="Create Application"
/>

3. Select a large model to view details
4. Copy the `API Password` from the authentication information in the upper right corner of the HTTP service interface

<img

src="https://oss.vidol.chat/assets/8669131e67e5276fe0744754ba4b1645.webp>"
alt="Copy API Key"
/>

## Step 2: Configure iFlytek Xinghuo in LobeVidol

1. Access the `Settings` interface in LobeVidol
2. Find the `iFlytek Xinghuo` settings under `Language Model`

<img

src="https://oss.vidol.chat/assets/4169b5d9f7534f9f89c8426445e9a080.webp>"
alt="Enter API Key"
/>

3. Enter the obtained API Key
4. Choose an iFlytek Xinghuo model for your AI assistant to start the conversation

<img

src="https://oss.vidol.chat/assets/4175fc55c2093d635f15a3287e89e977.webp>"
alt="Select iFlytek Xinghuo Model and Start Conversation"
/>

<Warning>During use, you may need to pay the API service provider, please refer to the relevant pricing policy of iFlytek Xinghuo.</Warning>

You can now use the models provided by iFlytek Xinghuo for conversations in LobeVidol.
