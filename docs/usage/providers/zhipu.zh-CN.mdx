[智谱AI](https://www.zhipuai.cn/) 是一家源自清华大学计算机系技术成果的高科技公司,成立于2019年,专注于自然语言处理、机器学习和大数据分析，致力于在认知智能领域拓展人工智能技术的边界。

本文档将指导你如何在 LobeVidol中使用智谱 AI:

## 步骤一：获取智谱 AI 的 API 密钥

1. 访问并登录 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 初次登录时系统会自动为你创建好 API 密钥并赠送 25M Tokens 的资源包
3. 进入顶部的 `API密钥` 可以查看你的 API

<img
 
  src="https://oss.vidol.chat/assets/61809c04dc49f8f59d61a7bc001b4c51.webp"
  alt="获得智谱AI API 密钥"
/>

## 步骤二：在 LobeVidol中配置智谱AI

1. 访问LobeVidol的`设置`界面
2. 在`语言模型`下找到`智谱AI`的设置项

<img
 
  src="https://oss.vidol.chat/assets/3645a17a19023fa730f1e7b4133dac8f.webp"
  alt="LobeVidol中填写智谱AI API 密钥"
/>

3. 打开智谱 AI 并填入获得的 API 密钥
4. 为你的助手选择一个智谱AI的模型即可开始对话

<img
 
  src="https://oss.vidol.chat/assets/ad84075d5a691dfc1135d496e6d6b618.webp"
  alt="选择并使用智谱AI模型"
/>

<Warning>在使用过程中你可能需要向 API 服务提供商付费，请参考智谱AI的费用政策。</Warning>

至此你已经可以在 LobeVidol中使用智谱AI提供的模型进行对话了。
