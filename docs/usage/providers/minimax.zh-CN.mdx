## 概述

[MiniMax](https://www.minimaxi.com/) 是 2021 年成立的通用人工智能科技公司，致力于与用户共创智能。MiniMax 自主研发了不同模态的通用大模型，其中包括万亿参数的 MoE 文本大模型、语音大模型以及图像大模型。并推出了海螺 AI 等应用。

## 配置步骤

### 1. 获取 MiniMax API 密钥

1. 注册并登录 [MiniMax 开放平台](https://www.minimaxi.com/platform)
2. 在 `账户管理` 中找到 `接口密钥` 菜单，并创建新的密钥

<img
  src="https://oss.vidol.chat/assets/8cc130b9a3a6e402abbe427281bf2c40.webp"
  alt="创建 MiniMax API 密钥"
/>

3. 填写一个 API 密钥的名称并创建

<img
  src="https://oss.vidol.chat/assets/2a40c12b958449ed89ca9efe043fc23d.webp"
  alt="填写 API 密钥名称"
/>

4. 在弹出的对话框中复制 API 密钥，并妥善保存

<img
  src="https://oss.vidol.chat/assets/611fae47d64cf80488d5ad215bf03040.webp"
  alt="保存 API 密钥"
/>

<Warning>
  请安全地存储密钥，因为它只会出现一次。如果您意外丢失它，您将需要创建一个新密钥。
</Warning>

### 2. 在 LobeVidol中配置 MiniMax

1. 访问 LobeVidol的`设置`界面
2. 在`语言模型`下找到`MiniMax`的设置项

<img
  src="https://oss.vidol.chat/assets/c23b00b27a6fb8abe69bef5fc8454245.webp"
  alt="填入 API 密钥"
/>

3. 打开 Minimax 并填入获得的 API 密钥
4. 为你的 AI 助手选择一个 MiniMax 的模型即可开始对话

<img
  src="https://oss.vidol.chat/assets/64fd5aa42ee44ebc248287c2a381da45.webp"
  alt="选择 MiniMax 模型并开始对话"
/>

<Warning>
  在使用过程中你可能需要向 API 服务提供商付费，请参考 MiniMax 的相关费用政策。
</Warning>

至此你已经可以在 LobeVidol中使用 MiniMax 提供的模型进行对话了。
