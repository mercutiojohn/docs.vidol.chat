# LobeVidolでの百川の使用

<img
src="https://oss.vidol.chat/docs/2024/12/88be132f5cc8c1cfb974545d11f7897f.png"
alt="LobeVidolでの百川の使用"
/>

この記事では、LobeVidolで百川を使用する方法を説明します：

<Steps>
  <Step title="百川スマートAPIキーの取得">
    - [百川スマート](https://platform.baichuan-ai.com/homePage)アカウントを作成
    - [APIキー](https://platform.baichuan-ai.com/console/apikey)を作成して取得

```
<img
  src="https://oss.vidol.chat/assets/217222e643d99ab3ba01fe92906f3314.webp"
  alt="APIキーの作成"
/>
```

  </Step>

  <Step title="LobeVidolでの百川の設定">
    - LobeVidolの`設定`画面にアクセス
    - `言語モデル`の下にある`百川`の設定項目を見つける

```
<img
  src="https://oss.vidol.chat/assets/8d4fbb776e2209a1ec58c6b3516351a1.webp"
  alt="APIキーの入力"
/>

- 取得したAPIキーを入力
- あなたのAIアシスタントのために百川のモデルを選択し、対話を開始

<img
  src="https://oss.vidol.chat/assets/2cfe64ead120815f7ba7100bc3dcfd48.webp"
  alt="百川モデルを選択して対話を開始"
/>
```

  </Step>
</Steps>

<Warning>使用中にAPIサービスプロバイダーに料金を支払う必要がある場合がありますので、百川の関連料金ポリシーを参照してください。</Warning>

これで、LobeVidolで百川が提供するモデルを使用して対話を行うことができるようになりました。
