## Aperçu

[SiliconCloud](https://siliconflow.cn/) est une plateforme de services d'intelligence artificielle basée sur des modèles fondamentaux open source, offrant divers services d'IA générative (GenAI).

Cet article vous guidera sur la façon d'utiliser SiliconCloud dans LobeVidol.

<img

src="https://oss.vidol.chat/docs/2024/12/f3fc8b92a6bbfa136f2f178afb830700.png"
alt="Bannière SiliconCloud"
/>

## Étapes de configuration

### 1. Obtenez la clé API de SiliconCloud

1. Inscrivez-vous et connectez-vous à [SiliconCloud](https://cloud.siliconflow.cn/account/ak)
2. Cliquez sur le menu `Clés API` à gauche
3. Créez une clé API et copiez-la

<img
src="https://oss.vidol.chat/assets/56e5331e7fae3754820790c824cdc480.webp"
alt="Créer une clé API"
/>

### 2. Configurez SiliconCloud dans LobeVidol

1. Accédez à l'interface `Paramètres` de LobeVidol
2. Sous `Modèle de langage`, trouvez l'option de configuration pour `SiliconFlow`
3. Entrez la clé API obtenue

<img
src="https://oss.vidol.chat/assets/94f55c97a24a08c7a5923c23ee2d7eef.webp"
alt="Entrer la clé API"
/>

### 3. Commencez à utiliser

Choisissez un modèle SiliconCloud pour votre assistant AI afin de commencer à dialoguer

<img
src="https://oss.vidol.chat/assets/94397c91265c37b9f313dc439b90125f.webp"
alt="Choisir un modèle SiliconCloud et commencer à dialoguer"
/>

<Warning>
  Pendant l'utilisation, vous pourriez avoir besoin de payer le fournisseur de services API, veuillez consulter la politique tarifaire de SiliconCloud.
</Warning>

Vous pouvez maintenant utiliser les modèles fournis par SiliconCloud pour dialoguer dans LobeVidol.
