Gemini AI est un ensemble de modèles de langage de grande taille (LLM) créé par Google AI, connu pour ses avancées de pointe dans la compréhension et le traitement multimodal. C'est essentiellement un puissant outil d'intelligence artificielle capable de traiter une variété de tâches impliquant différents types de données, et pas seulement du texte.

Ce document vous guidera sur la façon d'utiliser Google Gemini dans LobeVidol :

## Étape 1 : Obtenir une clé API de Google

1. Accédez et connectez-vous à [Google AI Studio](https://aistudio.google.com/)
2. Dans le menu `Obtenir une clé API`, sélectionnez `Créer une clé API`

<img
src="https://oss.vidol.chat/assets/9409068a65c0bf31c22927195c6705d0.webp"
alt="Générer une clé Google Gemini"
/>

3. Sélectionnez un projet et créez une clé API, ou créez une clé API dans un nouveau projet

<img
src="https://oss.vidol.chat/assets/6f335755e69b857938b5e14775cd6eff.webp"
alt="Entrer le nom de la clé API"
/>

4. Copiez la clé API dans la boîte de dialogue qui s'affiche

<img
src="https://oss.vidol.chat/assets/91529c43450e2039f36d1f276e696a9b.webp"
alt="Copier la clé API"
/>

## Étape 2 : Configurer OpenAI dans LobeVidol

1. Accédez à l'interface `Paramètres` de LobeVidol
2. Sous `Modèle de langage`, trouvez l'option de configuration pour `Google Gemini`

<img
src="https://oss.vidol.chat/assets/246cfb57701c6227c84fd77c5a71311b.webp"
alt="Remplir la clé API Google Gemini dans LobeVidol"
/>

3. Activez Google Gemini et entrez la clé API obtenue
4. Choisissez un modèle Gemini pour votre assistant afin de commencer la conversation

<img
src="https://oss.vidol.chat/assets/e1e578ecfc5cde9c4191f0241bad4fd3.webp"
alt="Choisir et utiliser un modèle Gemini"
/>

<Warning>Au cours de l'utilisation, vous pourriez avoir besoin de payer le fournisseur de services API, veuillez consulter la politique tarifaire de Google Gemini.</Warning>

Vous pouvez maintenant utiliser Google Gemini dans LobeVidol.
