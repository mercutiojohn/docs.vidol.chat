[Fireworks.ai](https://fireworks.ai/) est une plateforme d'inférence de modèle d'IA générative haute performance, permettant aux utilisateurs d'accéder et d'utiliser divers modèles via son API. La plateforme prend en charge plusieurs modalités, y compris les modèles de langage textuel et visuel, et offre des fonctionnalités telles que les appels de fonction et le format JSON pour améliorer la flexibilité du développement d'applications.

Cet article vous guidera sur la façon d'utiliser Fireworks AI dans LobeVidol.

## Étapes de configuration

### 1. Obtenir la clé API de Fireworks AI

1. Connectez-vous à la [console Fireworks.ai](https://fireworks.ai/account/api-keys)
2. Accédez à la page `Utilisateur`, puis cliquez sur `Clés API`
3. Créez une nouvelle clé API

<img
src="https://oss.vidol.chat/assets/d0c4369f894abb5ad6e514059b8f378e.webp"
alt="Créer une clé API"
/>

4. Copiez et enregistrez la clé API générée

<img
src="https://oss.vidol.chat/assets/89a8dadc85902334ce8d2d5b78abf709.webp"
alt="Enregistrer la clé API"
/>

<Warning>
  Veuillez stocker la clé en toute sécurité, car elle n'apparaîtra qu'une seule fois. Si vous la perdez accidentellement, vous devrez créer une nouvelle clé.
</Warning>

### 2. Configurer Fireworks AI dans LobeVidol

1. Accédez à l'interface `Paramètres` de LobeVidol
2. Sous `Modèle de langage`, trouvez l'option de configuration pour `Fireworks AI`

<img
src="https://oss.vidol.chat/assets/74f5de8ee68e57b472d1fc56a2df40d5.webp"
alt="Entrer la clé API"
/>

3. Saisissez la clé API obtenue
4. Choisissez un modèle Fireworks AI pour votre assistant IA afin de commencer à dialoguer

<img
src="https://oss.vidol.chat/assets/ba0243e75b0421b6dd7dadad02e4b0d6.webp"
alt="Choisir un modèle Fireworks AI et commencer à dialoguer"
/>

<Warning>
  Pendant l'utilisation, vous pourriez avoir à payer le fournisseur de services API, veuillez consulter la politique tarifaire de Fireworks AI.
</Warning>

Vous pouvez maintenant utiliser les modèles fournis par Fireworks AI pour dialoguer dans LobeVidol.
