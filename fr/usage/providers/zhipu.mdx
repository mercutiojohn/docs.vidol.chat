[Zhipu AI](https://www.zhipuai.cn/) est une entreprise de haute technologie issue des résultats de recherche du département d'informatique de l'Université Tsinghua, fondée en 2019. Elle se concentre sur le traitement du langage naturel, l'apprentissage automatique et l'analyse de grandes données, et s'efforce d'étendre les frontières de la technologie de l'intelligence artificielle dans le domaine de l'intelligence cognitive.

Ce document vous guidera sur la façon d'utiliser Zhipu AI dans LobeVidol :

## Étape 1 : Obtenir la clé API de Zhipu AI

1. Visitez et connectez-vous à la [plateforme ouverte de Zhipu AI](https://open.bigmodel.cn/)
2. Lors de votre première connexion, le système créera automatiquement une clé API pour vous et vous offrira un pack de ressources de 25M Tokens
3. Accédez à `Clé API` en haut pour voir votre API

<img

src="https://oss.vidol.chat/assets/61809c04dc49f8f59d61a7bc001b4c51.webp>"
alt="Obtenir la clé API de Zhipu AI"
/>

## Étape 2 : Configurer Zhipu AI dans LobeVidol

1. Accédez à l'interface `Paramètres` de LobeVidol
2. Sous `Modèle de langage`, trouvez l'option de configuration pour `Zhipu AI`

<img

src="https://oss.vidol.chat/assets/3645a17a19023fa730f1e7b4133dac8f.webp>"
alt="Remplir la clé API de Zhipu AI dans LobeVidol"
/>

3. Activez Zhipu AI et saisissez la clé API obtenue
4. Choisissez un modèle Zhipu AI pour votre assistant afin de commencer le dialogue

<img

src="https://oss.vidol.chat/assets/ad84075d5a691dfc1135d496e6d6b618.webp>"
alt="Choisir et utiliser le modèle Zhipu AI"
/>

<Warning>Au cours de l'utilisation, vous pourriez avoir besoin de payer le fournisseur de services API, veuillez consulter la politique tarifaire de Zhipu AI.</Warning>

Vous pouvez maintenant utiliser les modèles fournis par Zhipu AI pour dialoguer dans LobeVidol.
