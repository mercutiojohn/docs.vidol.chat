## Aperçu

[MiniMax](https://www.minimaxi.com/) est une entreprise de technologie d'intelligence artificielle générale fondée en 2021, dédiée à la co-création intelligente avec les utilisateurs. MiniMax a développé de manière autonome différents modèles de grande taille multimodaux, y compris un modèle de texte MoE avec un trillion de paramètres, ainsi que des modèles de voix et d'image. Ils ont également lancé des applications telles que l'IA Conch.

## Étapes de configuration

### 1. Obtenir la clé API MiniMax

1. Inscrivez-vous et connectez-vous à la [plateforme ouverte MiniMax](https://www.minimaxi.com/platform)
2. Dans `Gestion de compte`, trouvez le menu `Clé d'API` et créez une nouvelle clé

<img
src="https://oss.vidol.chat/assets/8cc130b9a3a6e402abbe427281bf2c40.webp"
alt="Créer une clé API MiniMax"
/>

3. Remplissez un nom pour la clé API et créez-la

<img
src="https://oss.vidol.chat/assets/2a40c12b958449ed89ca9efe043fc23d.webp"
alt="Remplir le nom de la clé API"
/>

4. Copiez la clé API dans la boîte de dialogue qui s'ouvre et conservez-la en lieu sûr

<img
src="https://oss.vidol.chat/assets/611fae47d64cf80488d5ad215bf03040.webp"
alt="Sauvegarder la clé API"
/>

<Warning>
  Veuillez stocker la clé en toute sécurité, car elle n'apparaîtra qu'une seule fois. Si vous la perdez accidentellement, vous devrez créer une nouvelle clé.
</Warning>

### 2. Configurer MiniMax dans LobeVidol

1. Accédez à l'interface `Paramètres` de LobeVidol
2. Sous `Modèle de langage`, trouvez l'option de configuration pour `MiniMax`

<img
src="https://oss.vidol.chat/assets/c23b00b27a6fb8abe69bef5fc8454245.webp"
alt="Entrer la clé API"
/>

3. Activez MiniMax et entrez la clé API obtenue
4. Choisissez un modèle MiniMax pour votre assistant AI afin de commencer la conversation

<img
src="https://oss.vidol.chat/assets/64fd5aa42ee44ebc248287c2a381da45.webp"
alt="Choisir un modèle MiniMax et commencer la conversation"
/>

<Warning>
  Pendant l'utilisation, vous pourriez avoir besoin de payer le fournisseur de services API, veuillez consulter la politique tarifaire de MiniMax.
</Warning>

Vous pouvez maintenant utiliser les modèles fournis par MiniMax pour converser dans LobeVidol.
