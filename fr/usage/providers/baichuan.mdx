# Utiliser Baichuan dans LobeVidol

<img
src="https://oss.vidol.chat/docs/2024/12/88be132f5cc8c1cfb974545d11f7897f.png"
alt="Utiliser Baichuan dans LobeVidol"
/>

Cet article vous guidera sur la façon d'utiliser Baichuan dans LobeVidol :

<Steps>
  <Step title="Obtenir la clé API de Baichuan">
    - Créez un compte sur [Baichuan](https://platform.baichuan-ai.com/homePage)
    - Créez et obtenez votre [clé API](https://platform.baichuan-ai.com/console/apikey)

```
<img
  src="https://oss.vidol.chat/assets/217222e643d99ab3ba01fe92906f3314.webp"
  alt="Créer une clé API"
/>
```

  </Step>

  <Step title="Configurer Baichuan dans LobeVidol">
    - Accédez à l'interface de `paramètres` de LobeVidol
    - Trouvez l'option de configuration de `Baichuan` sous `modèle de langage`

```
<img
  src="https://oss.vidol.chat/assets/8d4fbb776e2209a1ec58c6b3516351a1.webp"
  alt="Entrer la clé API"
/>

- Entrez la clé API obtenue
- Choisissez un modèle Baichuan pour votre assistant AI afin de commencer à dialoguer

<img
  src="https://oss.vidol.chat/assets/2cfe64ead120815f7ba7100bc3dcfd48.webp"
  alt="Choisir un modèle Baichuan et commencer à dialoguer"
/>
```

  </Step>
</Steps>

<Warning>Vous pourriez avoir besoin de payer le fournisseur de services API pendant l'utilisation, veuillez consulter la politique tarifaire de Baichuan.</Warning>

Vous pouvez maintenant utiliser les modèles fournis par Baichuan pour dialoguer dans LobeVidol.
