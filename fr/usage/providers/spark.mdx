[Xinghuo d’iFly](https://xinghuo.xfyun.cn/) est un puissant modèle d'IA lancé par iFlytek, doté de connaissances interdomaines et de capacités de compréhension linguistique, capable d'exécuter diverses tâches telles que des questions-réponses, des dialogues et des créations littéraires.

Cet article vous guidera sur la façon d'utiliser Xinghuo d’iFly dans LobeVidol.

## Étape 1 : Obtenir la clé API de Xinghuo

1. Inscrivez-vous et connectez-vous à la [plateforme ouverte d’iFly](https://console.xfyun.cn/)
2. Créez une application

<img

src="https://oss.vidol.chat/assets/4d0d191b487c114abf084eb7f2dc381c.webp>"
alt="Créer une application"
/>

3. Sélectionnez un modèle de grande taille pour voir les détails
4. Copiez le `API Password` dans les informations d'authentification de l'interface de service HTTP en haut à droite

<img

src="https://oss.vidol.chat/assets/8669131e67e5276fe0744754ba4b1645.webp>"
alt="Copier la clé API"
/>

## Étape 2 : Configurer Xinghuo d’iFly dans LobeVidol

1. Accédez à l'interface de `paramètres` de LobeVidol
2. Sous `modèle de langue`, trouvez l'option de configuration pour `Xinghuo d’iFly`

<img

src="https://oss.vidol.chat/assets/4169b5d9f7534f9f89c8426445e9a080.webp>"
alt="Entrer la clé API"
/>

3. Saisissez la clé API obtenue
4. Choisissez un modèle de Xinghuo d’iFly pour votre assistant IA afin de commencer le dialogue

<img

src="https://oss.vidol.chat/assets/4175fc55c2093d635f15a3287e89e977.webp>"
alt="Choisir un modèle de Xinghuo d’iFly et commencer le dialogue"
/>

<Warning>Au cours de l'utilisation, vous pourriez avoir besoin de payer le fournisseur de services API, veuillez consulter la politique tarifaire de Xinghuo d’iFly.</Warning>

Vous pouvez désormais utiliser le modèle fourni par Xinghuo d’iFly pour dialoguer dans LobeVidol.
