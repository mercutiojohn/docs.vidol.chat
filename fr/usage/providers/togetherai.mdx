[together.ai](https://www.together.ai/) est une plateforme spécialisée dans le domaine de l'intelligence artificielle générative (AIGC), fondée en juin 2022. Elle s'engage à construire une plateforme cloud pour exécuter, entraîner et affiner des modèles open source, offrant une capacité de calcul évolutive à des prix inférieurs à ceux des fournisseurs traditionnels.

Ce document vous guidera sur la façon d'utiliser Together AI dans LobeVidol :

## Étape 1 : Obtenir la clé API de Together AI

1. Visitez et connectez-vous à [Together AI API](https://api.together.ai/)
2. Lors de votre première connexion, le système créera automatiquement une clé API pour vous et vous offrira un crédit de 5,0 $

<img

src="https://oss.vidol.chat/assets/963ea14c00335f0e9369901d6dba79b2.webp>"
alt="Obtenir la clé API de together.ai"
/>

3. Si vous ne l'avez pas sauvegardée, vous pouvez également la consulter à tout moment ultérieur via l'interface `Clé API` dans `Paramètres`

<img

src="https://oss.vidol.chat/assets/47538a534659ed993831615752c23552.webp>"
alt="Consulter la clé API de together.ai"
/>

## Étape 2 : Configurer Together AI dans LobeVidol

1. Accédez à l'interface `Paramètres` de LobeVidol
2. Sous `Modèle de langage`, trouvez l'option de configuration pour `together.ai`

<img

src="https://oss.vidol.chat/assets/305a102705f5379f5d2b795793312ba6.webp>"
alt="Remplir la clé API de Together AI dans LobeVidol"
/>

3. Ouvrez together.ai et saisissez la clé API obtenue
4. Choisissez un modèle Together AI pour votre assistant afin de commencer à dialoguer

<img

src="https://oss.vidol.chat/assets/8ab9ae6ed6282bcdd6473b18d68a2eda.webp>"
alt="Choisir et utiliser un modèle Together AI"
/>

<Warning>Au cours de l'utilisation, vous pourriez avoir besoin de payer le fournisseur de services API, veuillez consulter la politique tarifaire de Together AI.</Warning>

Vous pouvez maintenant utiliser les modèles fournis par Together AI pour dialoguer dans LobeVidol.
