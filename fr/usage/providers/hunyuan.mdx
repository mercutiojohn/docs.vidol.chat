## Aperçu

[Tencent Hunyuan](https://hunyuan.tencent.com/) est un grand modèle lancé par Tencent, conçu pour fournir des services d'assistant intelligent aux utilisateurs. Il utilise des technologies de traitement du langage naturel pour aider les utilisateurs à résoudre des problèmes, à fournir des conseils et à générer du contenu. Les utilisateurs peuvent dialoguer avec le modèle pour obtenir rapidement les informations nécessaires, améliorant ainsi leur efficacité au travail.

Cet article vous guidera sur la façon d'utiliser Tencent Hunyuan dans LobeVidol.

## Étapes de configuration

### 1. Obtenir la clé API de Tencent Hunyuan

1. Inscrivez-vous et connectez-vous au [console de gestion Tencent Cloud](https://console.cloud.tencent.com/hunyuan/api-key)
2. Accédez à `Modèle Hunyuan` et cliquez sur `Gestion des clés API`
3. Créez une clé API

<img
src="https://oss.vidol.chat/assets/01dbe69cc092163f7ac782afb7d314c5.webp"
alt="Créer une clé API"
/>

4. Cliquez sur `Voir`, copiez la clé API dans le panneau contextuel et conservez-la en lieu sûr

<img
src="https://oss.vidol.chat/assets/856bd407c8a1510f616a4bdb1e02a883.webp"
alt="Sauvegarder la clé"
/>

### 2. Configurer Tencent Hunyuan dans LobeVidol

1. Accédez à l'interface `Paramètres` de LobeVidol
2. Sous `Modèle de langage`, trouvez l'option de configuration pour `Tencent Hunyuan`

<img
src="https://oss.vidol.chat/assets/bd8c97ef67055e3ff93c56e46c33fa8d.webp"
alt="Entrer la clé API"
/>

3. Saisissez la clé API obtenue
4. Choisissez un modèle Tencent Hunyuan pour votre assistant AI afin de commencer le dialogue

<img
src="https://oss.vidol.chat/assets/3c54d6f2d55fae843fbbfdc0bd7ffec7.webp"
alt="Choisir un modèle Tencent Hunyuan et commencer le dialogue"
/>

<Warning>
  Pendant l'utilisation, vous pourriez avoir besoin de payer le fournisseur de services API, veuillez consulter la politique tarifaire de Tencent Hunyuan.
</Warning>

Vous pouvez maintenant utiliser le modèle fourni par Tencent Hunyuan pour dialoguer dans LobeVidol.
