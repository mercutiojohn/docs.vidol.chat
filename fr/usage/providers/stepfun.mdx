[Stepfun Étoiles Sautantes](https://www.stepfun.com/) est une startup spécialisée dans le développement de l'intelligence artificielle générale (AGI), qui a récemment lancé le modèle de langage à 100 milliards de paramètres Step-1, le modèle multimodal à 100 milliards de paramètres Step-1V, ainsi que la version préliminaire du modèle de langage MoE à 1 trillion de paramètres Step-2.

Ce document vous guidera sur la façon d'utiliser Stepfun Étoiles Sautantes dans LobeVidol :

## Étape 1 : Obtenir la clé API de Stepfun Étoiles Sautantes

1. Visitez et connectez-vous à la [plateforme ouverte Stepfun Étoiles Sautantes](https://platform.stepfun.com/)
2. Accédez au menu `Clé API`, le système a déjà créé une clé API pour vous
3. Copiez la clé API créée

<img

src="https://oss.vidol.chat/assets/e15f56242cc4402902c33ac152ad0abd.webp>"
alt="Obtenir la clé API de Stepfun Étoiles Sautantes"
/>

## Étape 2 : Configurer Stepfun Étoiles Sautantes dans LobeVidol

1. Accédez à l'interface `Paramètres` de LobeVidol
2. Sous `Modèle de langage`, trouvez l'option de configuration pour `Stepfun Étoiles Sautantes`

<img

src="https://oss.vidol.chat/assets/95ffcabb47ddb989f8cb6e6d2ef451ad.webp>"
alt="Remplir la clé API"
/>

3. Activez Stepfun Étoiles Sautantes et entrez la clé API obtenue
4. Choisissez un modèle Stepfun Étoiles Sautantes pour votre assistant AI afin de commencer la conversation

<img

src="https://oss.vidol.chat/assets/771c3bd6028a014f3e0d3df01e4e4988.webp>"
alt="Choisir le modèle Stepfun Étoiles Sautantes et commencer la conversation"
/>

<Warning>Au cours de l'utilisation, vous pourriez avoir besoin de payer le fournisseur de services API, veuillez consulter la politique tarifaire de Stepfun Étoiles Sautantes.</Warning>

Vous pouvez maintenant utiliser les modèles fournis par Stepfun Étoiles Sautantes pour converser dans LobeVidol.
