## 配置步骤

### 步骤一：获取 Anthropic Claude API 密钥

1. 注册并创建一个 [Anthropic Claude API](https://www.anthropic.com/api) 账户
2. 在控制台中获取您的 [API 密钥](https://console.anthropic.com/settings/keys)

<Frame>
  <img 
    src="https://oss.vidol.chat/assets/ec6bd7ee7851ae4d9289dc9af7f73c1c.webp"
    alt="创建 API Key"
  />
</Frame>

<Warning>
  Claude API 目前提供 5 美元的免费积分，但仅适用于特定国家/地区。您可以在 Dashboard > Claim 中查看是否适用于您所在的地区。
  如果不适用，您也可以考虑使用其他支付方式。
</Warning>

3. 在 https://console.anthropic.com/settings/plans 中设置您的账单信息（选择"生成"计划以启用按量付费模式）

<Frame>
  <img 
    src="https://oss.vidol.chat/assets/66b784fffe7573242d86f4f73f177ce0.webp"
    alt="设置您的账单"
  />
</Frame>

### 步骤二：在 LobeVidol 中配置 Anthropic Claude

1. 打开 LobeVidol 的`设置`页面
2. 找到`AI 模型设置`部分下的`Anthropic Claude`配置项

<Frame>
  <img 
    src="https://oss.vidol.chat/assets/feb270dee460367739857f028f2f617e.webp"
    alt="填入 API 密钥"
  />
</Frame>

3. 输入您的 API 密钥
4. 从可用的 Anthropic Claude 模型中选择一个适合您需求的版本：
   - Claude 3 Opus：最强大的模型，适合复杂的视频脚本生成
   - Claude 3.5 Sonnet：平衡性能和成本的理想选择
   - Claude 3 Haiku：响应速度最快，适合快速创意构思

<Frame>
  <img 
    src="https://oss.vidol.chat/assets/4f9af602892219fa812b4321b1b2cdcd.webp"
    alt="选择 Anthropic Claude 模型并开始创作"
  />
</Frame>

<Warning>
  使用 API 服务需要支付相应费用，具体定价请参考 Anthropic Claude 的官方价格政策。建议在使用前详细了解各个模型的计费标准。
</Warning>

### 步骤三：开始使用

完成配置后，您就可以使用 Anthropic Claude 的强大功能来：

- 生成创意视频脚本
- 优化视频标题和描述
- 获取智能剪辑建议
- 生成字幕和配音文本
- 进行多语言内容创作

## 常见问题解答

<AccordionGroup>
  <Accordion title="如何选择合适的 Claude 模型？">
    - Claude 3 Opus：适合需要深度思考和复杂创意的项目
    - Claude 3.5 Sonnet：日常创作的最佳选择
    - Claude 3 Haiku：适合快速迭代和简单任务
  </Accordion>

  <Accordion title="API 密钥无法使用怎么办？">
    1. 确认账单信息是否配置正确
    2. 检查 API 密钥是否已激活
    3. 确认账户余额是否充足
    4. 检查是否存在地区限制
  </Accordion>

  <Accordion title="如何优化 API 使用成本？">
    1. 合理选择模型版本
    2. 使用更精确的提示词
    3. 批量处理相似任务
    4. 定期监控使用量
  </Accordion>
</AccordionGroup>
