Gemini AI是由 Google AI 创建的一组大型语言模型（LLM），以其在多模式理解和处理方面的尖端进步而闻名。它本质上是一个强大的人工智能工具，可以处理涉及不同类型数据的各种任务，而不仅仅是文本。

本文档将指导你如何在 LobeVidol中使用 Google Gemini:

## 步骤一：获取 Google 的 API 密钥

1. 访问并登录 [Google AI Studio](https://aistudio.google.com/)
2. 在 `获取 API 密钥` 菜单中 `创建 API 密钥`

<img
  src="https://oss.vidol.chat/assets/9409068a65c0bf31c22927195c6705d0.webp"
  alt="生成 Google Gemini 密钥"
/>

3. 选择一个项目并创建 API 密钥，或者在新项目中创建 API 密钥

<img
  src="https://oss.vidol.chat/assets/6f335755e69b857938b5e14775cd6eff.webp"
  alt="输入 API 密钥名称"
/>

4. 在弹出的对话框中复制 API 密钥

<img
  src="https://oss.vidol.chat/assets/91529c43450e2039f36d1f276e696a9b.webp"
  alt="复制 API 密钥"
/>

## 步骤二：在 LobeVidol中配置OpenAI

1. 访问LobeVidol的`设置`界面
2. 在`语言模型`下找到`Google Gemini`的设置项

<img
  src="https://oss.vidol.chat/assets/246cfb57701c6227c84fd77c5a71311b.webp"
  alt="LobeVidol中填写 Google Gemini API 密钥"
/>

3. 打开 Google Gemini 并填入获得的 API 密钥
4. 为你的助手选择一个 Gemini 的模型即可开始对话

<img
  src="https://oss.vidol.chat/assets/e1e578ecfc5cde9c4191f0241bad4fd3.webp"
  alt="选择并使用 Gemini 模型"
/>

<Warning>在使用过程中你可能需要向 API 服务提供商付费，请参考 Google Gemini 的费用政策。</Warning>

至此，你已经可以在 LobeVidol中使用 Google Gemini 啦。
