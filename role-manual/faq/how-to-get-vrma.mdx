---
title: '如何获取 .vrma 动画？'
description: '获取 VRM 动画资源的方法和平台介绍'
---

以下是一些获取 VRM 动画资源的网站和平台：

## 1. VRoid Hub

VRoid Hub 提供了一个"Photo Booth"功能，支持 VRM 动画的播放。

<Frame>
  <img src="https://cdn.nlark.com/yuque/0/2024/png/85418/1729957012345-abcd1234-5678-4321-9876-1234567890ab.png" alt="VRoid Hub Photo Booth" />
</Frame>

- 用户可以上传自己的 .vrma 文件并应用各种动画。
- 该平台还提供了一些免费的 VRM 动画文件供下载[1]。

## 2. BOOTH

BOOTH 是一个专注于创作者的市场，新增了"3D Motion/Animation"子类别。

<Frame>
  <img src="https://cdn.nlark.com/yuque/0/2024/png/85418/1729957123456-efgh5678-9012-3456-7890-abcdefghijkl.png" alt="BOOTH 3D Motion/Animation 类别" />
</Frame>

- 允许用户分享和销售 VRM 动画文件（.vrma）。
- 您可以在这里找到多种创作者的动画资源，包括免费的动画文件[1]。

## 3. VRM Posing Desktop

VRM Posing Desktop 是一个用于创建和导出 VRM 动画的软件。

<Frame>
  <img src="https://cdn.nlark.com/yuque/0/2024/png/85418/1729957234567-ijkl9012-3456-7890-1234-mnopqrstuvwx.png" alt="VRM Posing Desktop 界面" />
</Frame>

- 支持将 .vrma 文件导入并导出为视频格式（如 .mp4）。
- 您可以在 Steam 上购买该软件，并使用其强大的功能进行动画制作[2][4]。

## 4. GitHub VRM Specification

VRM Animation 的官方文档和规范可在 GitHub 上找到。

<Frame>
  <img src="https://cdn.nlark.com/yuque/0/2024/png/85418/1729957345678-mnop3456-7890-1234-5678-yzabcdefghij.png" alt="GitHub VRM Specification 页面" />
</Frame>

- 提供了关于如何使用 VRM 动画的详细信息。
- 包括如何导入和导出动画文件的指南[5]。

这些资源为您提供了丰富的工具和平台，以便获取和创建 VRM 动画。
