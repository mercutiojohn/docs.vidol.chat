# Using Tongyi Qianwen in LobeVidol

<img
src="https://oss.vidol.chat/docs/2024/12/83156a545234ce9264a97bb2991cfbbb.png"
alt="Using Tongyi Qianwen in LobeVidol"
/>

[Tongyi Qianwen](https://tongyi.aliyun.com/) is a large-scale language model independently developed by Alibaba Cloud, featuring strong natural language understanding and generation capabilities. It can answer various questions, create written content, express opinions, write code, and play a role in multiple fields.

This document will guide you on how to use Tongyi Qianwen in LobeVidol:

<Steps>
  <Step title="Activate DashScope Model Service">
    - Visit and log in to the Alibaba Cloud [DashScope](https://dashscope.console.aliyun.com/) platform
    - You need to activate the DashScope service upon your first visit
    - If you have already activated it, you can skip this step

<img
  src="https://oss.vidol.chat/assets/297d1f46136e9fe181e980e1f6cb4019.webp"
  alt="Activate DashScope Service"
/>

  </Step>

  <Step title="Obtain DashScope API Key">
    - Go to the `API-KEY` interface and create an API key

<img
  src="https://oss.vidol.chat/assets/2bdda9c970582f6ea4dfe30a66fef60d.webp"
  alt="Create Tongyi Qianwen API Key"
/>

- Copy the API key from the pop-up dialog and store it securely

<img
  src="https://oss.vidol.chat/assets/8b2e4629c9fa2b75d08b00de30490b59.webp"
  alt="Copy Tongyi Qianwen API Key"
/>

<Warning>
  Please store the key securely, as it will only appear once. If you accidentally lose it, you will need to create a new key.
</Warning>

  </Step>

  <Step title="Configure Tongyi Qianwen in LobeVidol">
    - Go to the `Settings` interface in LobeVidol
    - Find the settings for `Tongyi Qianwen` under `Language Model`

<img
  src="https://oss.vidol.chat/assets/ac3a3b2102f961ccc5b4d081a056cde3.webp"
  alt="Enter API Key"
/>

- Open Tongyi Qianwen and enter the obtained API key
- Choose a Qwen model for your AI assistant to start the conversation

<img
  src="https://oss.vidol.chat/assets/16e02355939ef75e9dc361050f2b7d5b.webp"
  alt="Select Qwen Model and Start Conversation"
/>

<Warning>
  You may need to pay the API service provider during use; please refer to the relevant pricing policy for Tongyi Qianwen.
</Warning>

  </Step>
</Steps>

You can now use the models provided by Tongyi Qianwen for conversations in LobeVidol.
