[360 Smart Brain](https://ai.360.com/) is a cognitive general model independently developed by 360 Company, aimed at providing powerful natural language processing capabilities for enterprises and developers. The model has been upgraded to version 4.0 and supports various application scenarios, including conversational services, image generation, vector database services, and more.

This article will guide you on how to use 360 Smart Brain in LobeVidol.

## Step 1: Obtain the 360 Smart Brain API Key

1. Register and log in to the [360 Smart Brain API Open Platform](https://ai.360.com/platform/keys)
2. Click on the `API Keys` menu on the left
3. Create an API key and copy it

<img
src="https://oss.vidol.chat/assets/688e6e10904ad46cf7f44bba6359f90c.webp"
alt="Create API Key"
/>

## Step 2: Configure 360 Smart Brain in LobeVidol

1. Go to the `Settings` interface in LobeVidol
2. Find the `360` settings under `Language Model`

<img
src="https://oss.vidol.chat/assets/65c86d6e63ddd5dd9896a6a67c054c0d.webp"
alt="Enter API Key"
/>

3. Enter the obtained API key
4. Choose a 360 Smart Brain model for your AI assistant to start the conversation

<img
src="https://oss.vidol.chat/docs/2024/12/e32889a1d501a14cba082ea42c7431f4.png"
alt="Select 360 Model and Start Conversation"
/>

<Warning>During use, you may need to pay the API service provider, please refer to the relevant pricing policy of 360 Smart Brain.</Warning>

You can now use the models provided by 360 Smart Brain for conversations in LobeVidol.
