# Using Wenxin Qianfan in LobeVidol

<Frame>
  <img
    src="https://oss.vidol.chat/docs/2024/12/9002ef959ed2146d4601aa733087253e.png"
    alt="Wenxin Qianfan Cover"
  />
</Frame>

[Wenxin Qianfan](https://qianfan.cloud.baidu.com/) is an artificial intelligence large language model platform launched by Baidu, supporting various application scenarios, including literary creation, commercial copy generation, mathematical logic reasoning, and more. The platform features cross-modal and cross-language deep semantic understanding and generation capabilities, widely used in search Q&A, content creation, and intelligent office fields.

This article will guide you on how to use Wenxin Qianfan in LobeVidol.

## Configuration Steps

### Step 1: Obtain the Wenxin Qianfan API Key

1. Register and log in to the [Baidu Intelligent Cloud Console](https://console.bce.baidu.com/)
2. Go to `Baidu Intelligent Cloud Qianfan ModelBuilder`
3. Select `Application Access` from the left menu
4. Create an application

<Frame>
  <img
    src="https://oss.vidol.chat/assets/f023683fe2f521e092cbdbebeb1de688.webp"
    alt="Create Application"
  />
</Frame>

5. In the user account menu, go to `Security Authentication` -> `Access Key` management page
6. Copy the `Access Key` and `Secret Key`, and keep them safe

<Frame>
  <img
    src="https://oss.vidol.chat/assets/2be6128378b48f17cd5acbe0f84dce77.webp"
    alt="Save Keys"
  />
</Frame>

### Step 2: Configure Wenxin Qianfan in LobeVidol

1. Access the `Settings` interface in LobeVidol
2. Find the `Wenxin Qianfan` settings under `Language Model`

<Frame>
  <img
    src="https://oss.vidol.chat/assets/831813e913d9a8f9e40c5562fe50d245.webp"
    alt="Enter API Key"
  />
</Frame>

3. Enter the obtained `Access Key` and `Secret Key`
4. Choose a Wenxin Qianfan model for your AI assistant to start the conversation

<Frame>
  <img
    src="https://oss.vidol.chat/assets/768ec6fd300785186b202437985857c4.webp"
    alt="Select Wenxin Qianfan Model and Start Conversation"
  />
</Frame>

<Warning>
  You may need to pay the API service provider during use; please refer to the relevant pricing policy of Wenxin Qianfan.
</Warning>

You can now use the models provided by Wenxin Qianfan for conversations in LobeVidol.
