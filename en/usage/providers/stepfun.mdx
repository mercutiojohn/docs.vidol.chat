[Stepfun Starry Steps](https://www.stepfun.com/) is a startup focused on the research and development of Artificial General Intelligence (AGI). It has launched the Step-1 100 billion parameter language model, the Step-1V 100 billion parameter multimodal model, and the preview version of the Step-2 trillion parameter MoE language model.

This document will guide you on how to use Stepfun Starry Steps in LobeVidol:

## Step 1: Obtain the Stepfun Starry Steps API Key

1. Visit and log in to the [Stepfun Starry Steps Open Platform](https://platform.stepfun.com/)
2. Go to the `API Keys` menu, where the system has already created an API key for you
3. Copy the created API key

<img

src="https://oss.vidol.chat/assets/e15f56242cc4402902c33ac152ad0abd.webp>"
alt="Obtain Stepfun Starry Steps API Key"
/>

## Step 2: Configure Stepfun Starry Steps in LobeVidol

1. Visit the `Settings` interface in LobeVidol
2. Find the settings for `Stepfun Starry Steps` under `Language Models`

<img

src="https://oss.vidol.chat/assets/95ffcabb47ddb989f8cb6e6d2ef451ad.webp>"
alt="Enter API Key"
/>

3. Open Stepfun Starry Steps and enter the obtained API key
4. Choose a Stepfun Starry Steps model for your AI assistant to start a conversation

<img

src="https://oss.vidol.chat/assets/771c3bd6028a014f3e0d3df01e4e4988.webp>"
alt="Select Stepfun Starry Steps model and start conversation"
/>

<Warning>During use, you may need to pay the API service provider, please refer to the relevant pricing policy of Stepfun Starry Steps.</Warning>

You can now use the models provided by Stepfun Starry Steps for conversations in LobeVidol.
