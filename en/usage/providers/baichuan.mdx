# Using Baichuan in LobeVidol

<img
src="https://oss.vidol.chat/docs/2024/12/88be132f5cc8c1cfb974545d11f7897f.png"
alt="Using Baichuan in LobeVidol"
/>

This article will guide you on how to use Baichuan in LobeVidol:

<Steps>
  <Step title="Obtain Baichuan Smart API Key">
    - Create an account on [Baichuan Smart](https://platform.baichuan-ai.com/homePage)
    - Create and obtain your [API Key](https://platform.baichuan-ai.com/console/apikey)

<img
  src="https://oss.vidol.chat/assets/217222e643d99ab3ba01fe92906f3314.webp"
  alt="Create API Key"
/>

  </Step>

  <Step title="Configure Baichuan in LobeVidol">
    - Go to the `Settings` page in LobeVidol
    - Find the `Baichuan` settings under `Language Model`

<img
  src="https://oss.vidol.chat/assets/8d4fbb776e2209a1ec58c6b3516351a1.webp"
  alt="Enter API Key"
/>

- Enter the obtained API Key
- Choose a Baichuan model for your AI assistant to start the conversation

<img
  src="https://oss.vidol.chat/assets/2cfe64ead120815f7ba7100bc3dcfd48.webp"
  alt="Select Baichuan model and start conversation"
/>
  </Step>
</Steps>

<Warning>During use, you may need to pay the API service provider, please refer to Baichuan's relevant pricing policy.</Warning>

You can now use the models provided by Baichuan for conversations in LobeVidol.
