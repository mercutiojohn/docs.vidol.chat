# Using DeepSeek in LobeVidol

<img
src="https://oss.vidol.chat/docs/2024/12/58a05d76bb89eeb59e238998b69c7059.png"
alt="Using DeepSeek in LobeVidol"
/>

[DeepSeek](https://www.deepseek.com/) is an advanced open-source large language model (LLM). The latest version, DeepSeek-V2, has significantly optimized architecture and performance, with training costs reduced by 42.5% and inference costs reduced by 93.3%.

This document will guide you on how to use DeepSeek in LobeVidol:

## Step 1: Obtain a DeepSeek API Key

1. First, you need to register and log in to the [DeepSeek](https://platform.deepseek.com/) open platform.

<Info>New users will receive a free quota of 500M Tokens.</Info>

2. Go to the `API keys` menu and click `Create API Key`.

<img
src="https://oss.vidol.chat/assets/498856598fdba60af6593408142db837.webp"
alt="Create DeepSeek API Key"
/>

3. Enter a name for your API key in the pop-up dialog.

<img
src="https://oss.vidol.chat/assets/89820732dcb092627c07d24143a37f60.webp"
alt="Enter DeepSeek API Key Name"
/>

4. Copy the generated API key and store it securely.

<img
src="https://oss.vidol.chat/assets/9f3d6287b883576a6b9cf67e28a6f43e.webp"
alt="Save DeepSeek API Key"
/>

<Warning>
  Please store the key securely, as it will only appear once. If you accidentally lose it, you will need to create a new key.
</Warning>

## Step 2: Configure DeepSeek in LobeVidol

1. Access the `App Settings` interface in LobeVidol.
2. Find the settings for `DeepSeek` under `Language Model`.

<img
src="https://oss.vidol.chat/assets/ee9802f91be4d6f1b686825b2433d75d.webp"
alt="Enter DeepSeek API Key"
/>

3. Enable DeepSeek and enter the API key you obtained.
4. Choose a DeepSeek model for your assistant to start chatting.

<img
src="https://oss.vidol.chat/assets/e84d84f68b6d6a8b3262dbcef2eded28.webp"
alt="Select DeepSeek Model"
/>

<Warning>
  You may need to pay the API service provider during use; please refer to DeepSeek's relevant pricing policy.
</Warning>

You can now use the models provided by DeepSeek for chatting in LobeVidol.
