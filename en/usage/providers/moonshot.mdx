# Using Moonshot AI in LobeVidol

<img

src="https://oss.vidol.chat/docs/2024/12/ef26c941f6d2b1d2617c7f6f93702569.png"
alt="Using Moonshot AI in LobeVidol"
/>

The Moonshot AI API is now available for everyone. This document will guide you on how to use [Moonshot AI](https://www.moonshot.cn/) in LobeVidol:

## Configuration Steps

### Step 1: Obtain Moonshot AI API Key

1. Apply for your [API key](https://platform.moonshot.cn/console/api-keys)

<img

src="https://oss.vidol.chat/assets/039f46519d8d785308768f01c190aed5.webp>"
alt="Apply for API Key"
/>

### Step 2: Configure Moonshot AI in LobeVidol

1. Go to the `Settings` interface in LobeVidol
2. Find the `Moonshot AI` settings under `Language Model`

<img

src="https://oss.vidol.chat/assets/46bfac50760257d60be12712a7a33083.webp>"
alt="Enter API Key"
/>

<Warning>
If you are using mistral.ai, your account must have a valid subscription for the API key to work properly. Newly created API keys may take 2-3 minutes to become active. If you click the "Test" button and it fails, please try again in 2-3 minutes.
</Warning>

3. Enter the obtained API key
4. Choose a Moonshot AI model for your AI assistant to start the conversation

<img

src="https://oss.vidol.chat/assets/5f412973b01225a1cbd3ae3b8f14aeef.webp>"
alt="Select Moonshot AI model and start conversation"
/>

<Warning>
During use, you may need to pay the API service provider, please refer to Moonshot AI's relevant pricing policy.
</Warning>

You can now use the models provided by Moonshot AI for conversation in LobeVidol.
